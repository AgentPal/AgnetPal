# PBC-004 Parallel Product Dev QA Review

## User Input

```text
Evaluate whether this feature should enter the next milestone from product, implementation, and QA angles.
```

## Expected Context Packet

Separate packets for product, implementation, and QA reviewer candidates.

Each packet forbids reading other reviewer drafts.

## Expected Workflow

Parallel Independent Review followed by summary stage.

## Expected Output

- independent final reports;
- summary of agreements, conflicts, evidence gaps, and next action;
- candidate language only.

## Failure Modes

- group chat style opinions;
- reviewers anchor on each other's drafts;
- fixed product/dev/QA route map;
- no conflict summary.

## Scoring Rubric

Score 0 / 1 / 2 for isolation, packet boundaries, independent output, conflict synthesis, and no fixed collaborators.
