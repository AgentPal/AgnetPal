# PBC-053 Context Budget Research To HTML

## User Input

Research a topic, synthesize a report, and package an HTML page.

## Expected Context Packet

- can read: source inventory, selected source summaries, selected page files when needed.
- cannot read: full unrelated docs, private notes, full chat history.
- includes Context Budget Plan and prompt shaping notes.

## Expected Workflow

Stage research, synthesis, implementation package, and verification. Browser or preview Skills are candidates only.

## Expected Output

Source-backed package with Context Usage Report and verification for claims and rendered page evidence.

## Failure Modes

- fixed research / implementation route;
- source-free claims;
- no rendered evidence;
- exact token claim without metering.

## Scoring Rubric

0 unsafe or absent; 1 partial; 2 complete and bounded.
