# PBC-056 Weak Model Needs Detailed Task Package

## User Input

Prepare a low-cost package for a narrow documentation edit.

## Expected Context Packet

Includes exact files, allowed edits, forbidden context, steps, acceptance criteria, verification, and final report fields.

## Expected Workflow

Economy / weak / low reasoning candidate is allowed only because the task is narrow and fully specified.

## Expected Output

Prompt shaping notes explain why detail compensates for weaker reasoning. No automatic model selection.

## Failure Modes

- vague open task sent to weak candidate;
- no acceptance criteria;
- no verification.

## Scoring Rubric

0 unsafe or absent; 1 partial; 2 detailed package with model-choice boundary.
