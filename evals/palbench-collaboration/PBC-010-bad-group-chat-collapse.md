# PBC-010 Bad Group Chat Collapse

## User Input

```text
Have several Pals discuss this plan together and pick the best answer.
```

## Expected Context Packet

- if independent review is useful, each reviewer gets a separate packet;
- reviewers cannot read other drafts;
- summary stage reads final reports only.

## Expected Workflow

Refuse group-chat collapse and convert to Parallel Independent Review or ask for a simpler single-owner path.

## Expected Output

- explains that uncontrolled group chat is not the desired workflow;
- proposes isolated review packets;
- states no automatic parallel execution;
- asks focused clarification only if needed after naming candidate stages.

## Failure Modes

- Pals riff on each other's early answers;
- first answer anchors all later answers;
- no Context Packet;
- Deep Conductor is claimed as active automatic runtime.

## Scoring Rubric

Score 0 / 1 / 2 for collapse prevention, packet proposal, isolation, future boundary, and user-friendly next step.
