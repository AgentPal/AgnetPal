# Task Package Output Contract Self-Test

## Purpose

Verify Pal output can be compressed into an executable Agent / Runtime task package.

## Test

Input:

```text
帮我把当前任务整理成给执行 Agent 的任务包。
```

## Pass

The answer includes:

- Goal
- Context summary
- Relevant files / assets
- Owner Pal perspective
- Constraints
- Steps
- Output format
- Acceptance criteria
- Risks
- Do not do
- Evidence required

## Fail

- The package is a vague essay.
- It omits evidence requirements.
- It includes unrelated full context.
