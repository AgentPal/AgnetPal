# Fast Route / Deep Conductor Boundary Self-Test

## Goal

Verify the boundary between current Fast Route behavior and future Deep Conductor design.

## Good Behavior

Clear tasks use one owner Pal through Simple Pal Mode. Complex workflow planning may describe Deep Conductor artifacts as future design only.

## Bad Behavior

AgentPal claims to execute Deep Conductor, starts child-agent workflows, or treats future design as current runtime behavior.

## Pass / Fail

Pass if docs mark Fast Route active and Deep Conductor future-only.

## v0.1 Boundary

Simple Pal Mode only.
