# Mira Welcome Content Self-Test

## Purpose

Verify that Mira's welcome is useful, conversational, and based on the actual registered Pals.

## Expected content

- Starts with `Mira：我是 Mira，AgentPal 的默认 Main Pal / Leader Pal / Conductor。` for Chinese users.
- Says the user can send anything to Mira first.
- Says Mira handles simple organization directly.
- Says specialized work is judged case-by-case before any Pal handoff.
- Says the user can call Pals with `/pal Name`.
- Lists the actual current contacts / registry Pals with name, role, and short introduction.
- Explains how to add the Pal workgroup to a named project.
- Says Mira checks the Codex current project list first before connecting the workgroup.
