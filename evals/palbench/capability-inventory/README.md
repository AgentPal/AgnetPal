# Capability Inventory PalBench Cases

This directory contains public-safe regression cases for Capability Inventory governance.

Cases here are Markdown eval descriptions only. They do not run scanners, validators, automatic discovery, automatic routing, external writes, or credential checks against live systems.

Use these cases to review whether docs, templates, examples, and metadata preserve the manual no-code Capability Inventory boundary.

Recent Business System cases cover manual profile creation, project-record references, review packets, manual update evidence packs, and manual writeback replay records. They must preserve no connector, no credential, no automatic scan, no automatic writeback, no automatic rollback, no external `.agentpal/` pollution, no central roster mutation, and no keyword routing.
