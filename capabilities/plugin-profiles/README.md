# Plugin Profiles

Plugin profiles describe plugin capabilities, permissions, limits, risk, and evidence requirements.

AgentPal does not automatically inspect, install, or call plugins as part of the Pal layer. A plugin profile is a judgement input only.

## Rules

- Do not claim the user has a plugin installed unless current Runtime evidence proves it.
- Do not treat plugins as Pals.
- Do not store private credentials, endpoint details, or private workspace data.
- Plugin use must remain permission-aware and evidence-backed.
