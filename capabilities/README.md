# Capability Inventory Compatibility Pointer

This root directory is kept only as a temporary compatibility pointer after R78.

Current sources:

- Standards and matrices: `standards/capability-inventory/`
- Example profiles: `examples/capability-inventory/`
- Current organization records: `workspace/organization/capability-inventory/`
- R78 migration evidence: `archive/migration-from-v0.3/root-legacy/capability-inventory/`

Capability Inventory is a no-code profile layer. It does not automatically scan runtimes, models, Skills, plugins, MCP servers, or business systems. It does not provide keyword routing, domain routing, or deterministic Pal assignment.

Do not copy this directory into external user projects. External projects use thin binding only.
