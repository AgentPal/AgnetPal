# Capability Profiles

This directory holds public-safe capability profile notes and illustrative examples.

Capability profiles describe possible runtimes, models, Skills, plugins, MCP servers, and Pal capabilities as judgement inputs. They are not route maps and not automatic environment probes in AgentPal v0.1.0-rc.1.

## Rules

- Examples are illustrative only.
- Do not treat a profile as proof that the user's environment has that capability.
- Do not encode task/domain -> fixed Pal or runtime choices.
- Do not store API keys, tokens, credentials, private paths, or private project data.
- Use profiles to support AI Judgement, Context Access Lists, Task Packages, verification, and PalBench.

## Subdirectories

| Directory | Purpose |
| --- | --- |
| `runtime-profiles/` | Runtime capability examples |
| `model-profiles/` | Model capability examples |
| `skill-profiles/` | Skill capability examples |
| `plugin-profiles/` | Plugin capability examples |
| `mcp-profiles/` | MCP capability examples |

AgentPal v0.1 active runtime remains Simple Pal Mode only.
