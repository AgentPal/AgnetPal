# Official Pal History Archive

This directory records that historical Pal reports, old runtime state snapshots, and old Mira memory records were moved out of `official/pals/` during R69 cleanup.

Reason:

- `official/pals/` should hold current public Pal Pack assets, templates, public placeholders, and examples.
- Runtime state and historical inspection reports should not be active Pal entry material.
- Public traceability is still useful, but the public release package should not retain historical runtime state or memory bodies.

Moved categories:

- historical Mira review and self-health reports were removed from active Pal paths
- PalSmith historical gap report was removed from active Pal paths
- Morgan and Nova historical PalSmith review reports were removed from active Pal paths
- old Mira and PalSmith runtime state files were removed from the public package
- old Mira memory records were removed from the public package

The archive is not an active runtime state location.
