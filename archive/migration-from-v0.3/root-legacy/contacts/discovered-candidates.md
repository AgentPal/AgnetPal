# Discovered Candidates

This file records directories that may be Pals but do not yet meet the Pal Pack contact requirements.

Current status: no candidates scanned.

Candidate directories must not enter contacts until they have `PAL.md`, `SKILL.md`, `pal.json`, type `pal-pack`, display name, aliases, and collaboration permissions.

## Candidate handling

For each candidate, record:

- directory name
- why it was not accepted
- missing files or invalid metadata
- whether the name follows `Name-role`
- suggested next step

Do not modify the candidate unless the user asks to repair that Pal Pack.
