# Plugins Compatibility Pointer

This root directory is kept only as a temporary compatibility pointer after R78.

Current sources:

- Plugin discovery standard: `standards/capability-inventory/plugin-scan-protocol.md`
- Current Skill, plugin, and MCP records: `workspace/organization/capability-inventory/`
- Plugin usage memory: `workspace/organization/capability-inventory/usage-memory/`
- Plugin and MCP example profiles: `examples/capability-inventory/`

AgentPal does not automatically inspect plugin state or enable tools. Availability must come from the current host runtime or an explicitly maintained central record.

Do not copy this directory into external user projects. External projects use thin binding only.
