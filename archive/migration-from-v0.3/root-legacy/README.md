# Root Legacy Paths

This directory contains root-level compatibility paths moved during v0.4 local structure cleanup.

These files are retained for migration traceability only. They are not current sources of truth.

Current sources:

- Official Pal Packs: `official/pals/`
- Central Pal contacts: `workspace/organization/contacts/`
- Compatibility registry records: `workspace/resources/registry/`
- Project binding templates: `templates/project-binding/`

Do not copy these legacy folders into external user projects.
