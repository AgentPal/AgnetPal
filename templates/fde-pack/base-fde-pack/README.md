# Base FDE Pack Template

This is a public-safe template for an FDE / Expert Delivery Pack.

Use it to package expert delivery knowledge, reusable checklists, delivery scope, update rules, and verification criteria without turning the pack into software or customer-private storage.

## Files

- `FDE.md`
- `fde.json`
- `expert-profile.md`
- `delivery-scope.md`
- `reusable-assets.md`
- `customer-private-boundary.md`
- `update-policy.md`
- `verification-checklist.md`

## Boundary

This template is no-code. It is not a connector, scanner, validator, marketplace, installer, runtime, or external project writer.

Do not add credentials, customer-private data, customer records, real contracts, real financial records, sensitive personal information, keyword route maps, or automatic execution claims.
