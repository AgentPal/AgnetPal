# Practical Org Pack

## Purpose

This Org Pack template describes a reusable no-code AI organization package.

It can be adapted into public-safe organization patterns, delivery packs,
practice packs, or team-operation packages while keeping customer-private facts
out of the reusable package.

## Intended Use

Use this package when a repeatable organization pattern needs:

- reusable role recommendations;
- workflow and runbook candidates;
- governance checkpoints;
- Capability Inventory references;
- verification checklists;
- clear private-asset boundaries.

## Recommended Pals

Recommended Pals are AI judgement inputs only. They are not routes, hard
assignments, or central roster edits.

The current Brain / AI must still read the central roster, user request, project
context, available evidence, risk, and this Org Pack before choosing a Pal or
Runtime candidate.

## Reusable Assets

Reusable assets belong in this Org Pack when they are de-identified,
public-safe, and useful across organizations.

## Customer-Private Assets

Customer-private data, credentials, private project memory, customer system
identifiers, private evidence, and customer-specific business rules belong in
approved private records, not this reusable package.

## Runtime Boundary

This Org Pack does not execute work. Host runtimes perform real file, command,
or external-system actions only when separately instructed and when evidence is
available.
