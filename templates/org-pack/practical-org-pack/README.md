# Practical Org Pack Template

This is a reusable no-code practical Org Pack template for AgentPal v0.5.

Use it to create a public-safe organization package with roles, workflows,
runbooks, governance, Capability Inventory references, verification, and private
boundary policy.

This template is not a Pal Pack, project directory, installer, marketplace
program, connector, scanner, validator, runtime, auto-sync engine, or
auto-execution engine.

## Files

- `ORG.md`
- `org.json`
- `roles/README.md`
- `workflows/README.md`
- `runbooks/README.md`
- `governance/README.md`
- `capability-inventory/README.md`
- `verification/README.md`
- `private-boundary.md`

## Boundary

- Do not copy the central Pal roster.
- Do not copy official Pal bodies.
- Do not modify `workspace/organization/contacts/pals.json`.
- Do not create keyword routes, domain routes, role routes, or fixed Pal
  assignments.
- Do not store credentials, private tokens, real customer data, or private
  project evidence.
- Do not write this Org Pack into an external project `.agentpal/` directory by
  default.
- Treat recommended Pals, Pal Skills, workflows, runbooks, and capability
  references as AI judgement inputs only.
