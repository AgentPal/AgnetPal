# Project Binding Templates

These templates are for thin AgentPal bindings in external user projects.

Use:

- `generic-codex/` for Codex-style or generic `AGENTS.md` runtimes.
- `claude-code/` for Claude Code projects.
- `prompts/` for install/update prompt text.

Default templates must not include `.agentpal/memory`, `.agentpal/state`, `.agentpal/reports`, `.agentpal/context`, `.agentpal/index`, `.agentpal/pals`, `.agentpal/workflows`, `.agentpal/evals`, `.agentpal/capability-inventory`, `.agentpal/business-systems`, `.agentpal/reviews`, `.agentpal/evidence`, `.agentpal/replay`, `.agentpal/rollback`, `.agentpal/verification`, `.agentpal/audit-trail`, `.agentpal/governance-decisions`, or `.agentpal/change-ledger`.
