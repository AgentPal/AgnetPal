# Agent Task Package Template

```text
Task Package
- Goal:
- Context summary:
- Relevant files / assets:
- Asset loading note:
- Owner Pal perspective:
- Constraints:
- Steps:
- Output format:
- Acceptance criteria:
- Risks:
- Do not do:
- Evidence required:
```

## Use

Use when a Pal needs to hand a compact task to a bottom-layer Agent / Runtime or when the user asks for an actionable execution package.

Use the Asset loading note to distinguish index-known paths from content-read files when that matters for execution or review.
