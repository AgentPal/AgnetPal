# Official AgentPal Assets

This directory contains official AgentPal organization assets that belong to the central AgentPal workspace.

- `pals/`: bundled official Pal Packs.
- `teams/`: official Pal team definitions and future team templates.
- `workflows/`: official no-code workflow descriptions and Task Package patterns.
- `org-packs/`: reusable organization packs.

External user projects should not receive copies of these assets by default. Thin project bindings point back to this workspace instead.
