# Skill Memory

This directory is reserved for reusable learning for Quinn: repeated task flows, useful work patterns, candidate Skills, Runbooks that should be promoted, and success/failure notes.

This is not a place to store third-party Skill source code, copied upstream documentation, user private data, credentials, or sensitive project material.

This directory is only for runtime notes before formal Skill creation, or for exception cases where required inputs are missing, content is unsafe/private, or a high-risk write needs approval. If the user explicitly asks to save a Skill, or similar operations happen more than 3 times, create the formal Skill under this Pal's own skills/<skill-id>/SKILL.md and update skills/index.md.

Official public Pal Pack releases keep only this README.md and .gitkeep; real runtime learning stays local and protected by .gitignore.
