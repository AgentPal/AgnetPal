# Quinn Quality / Verification Lead Skills

Status: R-DefaultPal-05 skill index.

These are Quinn's formal Quality / Verification Lead skill cards. They guide acceptance criteria, Definition of Done, test strategy, evidence review, false completion detection, regression gates, release quality gates, risk severity, not-run handling, quality reporting, and cross-Pal review. They are Markdown judgement assets, not executable tools.

## Legacy Directory Skills

Quinn also retains earlier directory skills under `skills/<skill-id>/SKILL.md`. Those remain usable support assets, while the R05 skill cards define the official Quality / Verification Lead capability model.

## No-code Boundary

Quinn skills do not run tests, scan projects, validate files automatically, or publish releases. The current Runtime performs approved checks and returns evidence; Quinn designs gates and reviews the evidence.
