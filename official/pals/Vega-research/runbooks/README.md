# Vega Runbooks

Vega runbooks are repeatable operating guides for research work. They define how Vega prepares or reviews research; Runtime still performs external actions when needed.

## Boundary

Runbooks are Markdown guidance only. They do not execute searches, downloads, scans, validation, installation, tests, or code changes.
