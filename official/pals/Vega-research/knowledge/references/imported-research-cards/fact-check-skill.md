# Fact Check Skill

## Source

- URL: https://github.com/petar-nauka/fact-check-skill
- Resource type: Skill
- License / usage boundary: MIT, per R2 GitHub metadata check on 2026-06-22.
- Current handling: public-card, convert-to-runbook, convert-to-knowledge-card.

## Main Use

Fact-checking and disinformation-detection methods for breaking claims into checkable units, comparing sources, and reporting uncertainty.

## Useful For Vega

- Claim decomposition.
- Source triangulation.
- Evidence strength notes.
- Confidence and uncertainty language.
- Fact-check report structure.

## Not Directly Used

- Vega does not become a final truth arbiter.
- Vega does not copy source code or generated card templates.
- Vega does not run third-party tools automatically.

## Risk Boundary

- Fact-checking can overstate certainty if sources are incomplete.
- A fact-check card is not a legal, medical, financial, or policy final judgment.
- Current facts still require fresh source checks.

## Public Package Boundary

- Contains third-party source code: no.
- Contains long third-party text: no.
- Enters Pal contacts: no.

## Recommended Trigger

Use as a reference when a research task asks whether a public claim is true, current, misleading, unverified, or source-dependent.

## Vega Handling

Vega may use this card as inspiration for its own fact-checking workflow, evidence table, and uncertainty labels. Any reusable method must be expressed in Vega's own Runbook or Knowledge Card.
