# Academic Source Review

学术资料不同于普通网页。Vega 处理论文、预印本、学术报告或引用资料时，应把来源信息、研究限制和可复用结论分开。

## Required Metadata

尽量记录：

- 标题。
- 作者。
- 年份。
- 期刊、会议、预印本平台或机构。
- DOI、arXiv ID 或稳定链接，如果可得。
- 发表时间、更新时间或访问时间。
- 研究领域。

## Separate These Layers

- 论文结论：作者在论文中主张什么。
- 证据和方法：样本、实验、数据、评估方式。
- 作者观点：解释、讨论或推测。
- 二手解读：别人如何理解这篇论文。
- Vega 推断：Vega 根据用户目标做出的适配判断。

## Risks

- 论文可能过时。
- 预印本可能未同行评审。
- 研究结论可能只适用于特定样本、任务、语言、地区或版本。
- 二手解读可能夸大。
- 摘要不能替代全文审读。

## Storage Boundary

Vega 不复制论文正文、图表、付费全文或大段引用。可以保存引用信息、短摘要、可用结论、适用范围、不确定性和后续查证建议。

## Output Template

```text
## 引用信息
## 研究问题
## 作者结论
## 方法和证据
## 可用于 AgentPal 的部分
## 不确定性和局限
## 是否适合知识卡
```
