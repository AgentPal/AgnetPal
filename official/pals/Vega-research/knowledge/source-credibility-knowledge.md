# source-credibility-knowledge

## Concept Explanation

Source credibility judges whether a source is fit for a specific claim. It includes source type, authority, currency, relevance, accuracy, purpose, evidence trail, and bias.

## Judgement Standards

- Official or primary sources are strongest for their own policies, specs, pricing, or releases.
- Academic and standards sources are strong for method and evidence-based fields.
- Community evidence can show experience patterns but needs corroboration.
- Marketing pages are weak for competitor claims.
- Freshness matters for prices, APIs, models, laws, schedules, and active projects.

## Examples

- Vendor docs support an API parameter claim.
- A university library guide supports a source evaluation method.

## Counterexamples

- A blog ranking supports "best overall" without criteria.
- A stale page supports a current pricing claim.

## Scope

Use for all source reviews and evidence matrix confidence.

## Common Mistakes

Treating source existence as support, ignoring bias, and failing to cross-check.
