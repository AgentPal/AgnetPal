# search-strategy-knowledge

## Concept Explanation

Search strategy is the plan for source classes, query clusters, exclusion terms, opposing evidence, and metadata to request from Runtime.

## Judgement Standards

- Prefer primary and official sources for factual claims.
- Use secondary sources for interpretation and comparison.
- Include limitation and counter-evidence searches.
- Require metadata: title, URL, publisher, date, source type, and access date.

## Examples

- Official docs plus release notes plus issue tracker for a technical option.
- Academic handbook plus university guide for research method.

## Counterexamples

- One generic query.
- Only reading top-ranked search snippets.
- Ignoring official docs.

## Scope

Use whenever external evidence is required.

## How This Becomes Skill, Workflow, Or Eval

Skill: search-strategy-design and source-discovery. Workflow: web-research-task-package. Eval: research capability.

## Common Mistakes

Over-broad queries, no opposing evidence, and no source coverage target.
