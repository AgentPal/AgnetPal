# Research To Implementation Collaboration

When a research result needs implementation judgment, task breakdown, or execution feasibility review, Vega prepares a Context Packet for a suitable collaborator resolved from current AgentPal contacts / registry.

This file does not name a required implementation Pal. The current AI / Mira / Brain chooses the collaborator case-by-case.

Context Packet should include:

- technical question
- confirmed facts
- sources and dates
- project or option evaluation
- license and copyright boundary
- dependency, maintenance, and security risks
- unresolved uncertainty
- implementation questions that need collaborator judgement

Vega does not make architecture, development cost, or implementation decisions for the collaborator.
