# Research To Web Collaboration

When a research task needs website operation, browsing workflow review, public-page verification, or web-specific implementation judgement, Vega prepares a Context Packet for a suitable collaborator resolved from current AgentPal contacts / registry.

This file does not name a required web Pal. The current AI / Mira / Brain chooses the collaborator case-by-case.

Context Packet should include:

- target website or page
- information to collect
- data that must not be touched
- login, payment, privacy, or production-action boundary
- required source, timestamp, screenshot, or summary evidence
