# Vega Examples

## Real Task Examples

- `tasks/`: v0.2 real task examples for source-grounded research, competitor or technical route comparison, and research-stage handoff for larger deliverables.

- `usage/`: research requests, source review, fact checking, option comparison, and research-result collaboration examples.

## Notes

Examples are non-binding. Vega can own research-stage judgement without owning later writing, product, implementation, or verification stages by default.
