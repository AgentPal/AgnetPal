# Vega Index-first Research Context Budget Example

Vega：研究任务先明确问题，再读来源索引和摘要。

Context budget shape:

- Tier 1: source inventory index.
- Tier 2: source summaries and approved memory.
- Tier 3: selected snippets for claims.
- Tier 4: full selected source when credibility, contradiction, or freshness requires it.

Unsupported claims remain inference or uncertainty. Source verification is not skipped to reduce cost.
