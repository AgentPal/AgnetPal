# evidence-matrix-workflow

## Trigger

Research will support a decision, recommendation, comparison, quality review, or knowledge update.

## Inputs

Claims, source inventory, credibility notes, contradictions, user criteria, and open questions.

## Steps

1. Convert conclusions into claim rows.
2. Attach evidence and Source IDs.
3. Assign confidence with rationale.
4. Add contradictions and open questions.
5. Suggest next research only where it changes the answer.
6. Use the matrix to constrain final synthesis.

## Decision points

- Is a claim directly supported?
- Is confidence high enough for recommendation?
- Are contradictions material?

## Outputs

Evidence matrix with claim, evidence, Source ID, confidence, contradictions, open questions, and next research.

## Quality checks

No unsupported factual claims; no hidden contradictions; no recommendation from low-confidence evidence without warning.

## Required user confirmation

Ask before using low-confidence evidence for high-impact action.

## Evidence to return

Matrix rows, source IDs, confidence rationale, and unresolved questions.
