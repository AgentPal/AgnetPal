# competitor-analysis

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：竞品分析

## 用途

帮助用户比较同类产品、项目、工具或方案的定位、能力、差异、可学习点和不应照搬点。

## 适用场景

- 分析某个产品和 AgentPal 的区别。
- 找类似工具。
- 比较功能、用户、门槛和生态。
- 为合适产品协作对象 提供资料。

## 不适合场景

- 投资级市场尽调。
- 需要商业机密或内部数据。
- 只看官网营销材料。

## 输入信息

- 竞品对象。
- 比较目标。
- 目标用户。
- 需要关注的维度。
- 来源材料。

## 处理流程

1. 区分官方介绍、第三方资料和用户反馈。
2. 提取目标用户、场景、功能范围、使用门槛、部署方式和商业模式。
3. 分析优点、缺点、生态能力和风险。
4. 对比 AgentPal 或用户项目。
5. 输出可学习点和不应照搬点。

## 输出格式

```text
# 竞品分析
## 对象
## 一句话定位
## 核心能力
## 相同点
## 不同点
## 可学习点
## 不应照搬点
## 结论
```

## 验收标准

- 不只复述官网。
- 至少区分官方自述和第三方/用户反馈。
- 明确适用范围和不确定性。
- 给出可行动下一步。

## 风险边界

不提供投资、法律或商业决策最终结论。不把未经证实的用户反馈当事实。
