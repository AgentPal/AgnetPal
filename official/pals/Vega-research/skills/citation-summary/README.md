# citation-summary

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：引用摘要

## 用途

把多个来源的关键事实整理成可引用摘要，保留来源、时间、支持程度和局限。

## 适用场景

- 写研究报告。
- 更新知识卡。
- 比较多个方案。
- 需要把来源交给其他 Pal。

## 不适合场景

- 保存全文。
- 复制长篇付费内容。
- 来源不明或不可引用。

## 输入信息

- 来源标题。
- URL 或出处。
- 发布时间 / 更新时间 / 访问时间。
- 关键事实摘要。
- 支持的结论。

## 处理流程

1. 提取关键事实。
2. 标注来源和时间。
3. 判断支持程度。
4. 记录局限和不确定性。
5. 避免复制版权全文。

## 输出格式

```text
## 关键事实
事实：
来源：
时间：
支持程度：
备注：
```

## 验收标准

- 每条事实都有来源。
- 摘要短而清楚。
- 长引用和全文复制被避免。
- 时间和局限被标注。

## 风险边界

引用摘要不等于取得再分发授权。不要保存版权资料全文。

## 与其他 Skill / Runbook 的关系

服务 `research-brief-writer`、`knowledge-card-curator` 和 handoff。
