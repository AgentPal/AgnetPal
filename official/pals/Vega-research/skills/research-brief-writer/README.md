# research-brief-writer

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：研究简报写作

## 用途

把调研结果压缩成结论先行、证据清楚、能支持下一步行动的研究简报。

## 适用场景

- 技术选型简报。
- 开源项目评估报告。
- 竞品分析摘要。
- 给 suitable implementation collaborator 的资料结论。

## 不适合场景

- 证据不足却要求强结论。
- 对外发布的正式报告未经人工审查。
- 高风险专业最终建议。

## 输入信息

- 研究问题。
- 证据表。
- 来源摘要。
- 不确定项。
- 用户目标。
- 后续接收方。

## 处理流程

1. 先写结论。
2. 说明背景和问题。
3. 汇总主要发现。
4. 列出证据来源和不确定点。
5. 写风险、建议和下一步。
6. 生成知识卡候选。

## 输出格式

```text
# Research Brief
## 结论先行
## 背景
## 主要发现
## 证据来源
## 不确定点
## 风险
## 建议
## 可沉淀知识卡
## 下一步
```

## 验收标准

- 结论和证据对应。
- 不确定项单独列出。
- 建议可执行。
- 适合交给下一步 Pal 或用户决策。

## 风险边界

研究简报不能伪装成最终专业意见。对外发布前需要人工确认来源和版权边界。

## 与其他 Skill / Runbook 的关系
