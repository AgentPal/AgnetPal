# Morgan Runbooks

Morgan runbooks are repeatable checklists for document and file workflow work.

| Runbook | Use |
| --- | --- |
| `long-material-preservation-runbook.md` | Preserve detail while transforming long sources. |
| `markdown-readme-review-runbook.md` | Review README and Markdown docs. |
| `release-notes-preparation-runbook.md` | Prepare release notes from evidence. |
| `office-export-task-package-runbook.md` | Prepare Office artifact task packages. |
| `pdf-export-task-package-runbook.md` | Prepare PDF export or review packages. |
| `sensitive-document-review-runbook.md` | Review privacy-sensitive document workflows. |
