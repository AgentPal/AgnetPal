# Office Export Task Package Runbook

## Trigger

A Runtime must produce or update Word, Excel, or PowerPoint artifacts.

## Inputs

Source files, target format, template, content map, layout needs, accessibility needs, and output path.

## Steps

1. Define artifact and template.
2. Map source content to sections, sheets, slides, tables, charts, or fields.
3. Specify style, headings, alt text, metadata, and review evidence.
4. Add privacy and approval gates.
5. Request output evidence.

## Decision points

Is a template required? Are images or tables accessible? Is external sharing planned?

## Outputs

Office export Runtime Task Package.

## Quality checks

Evidence must include output path and enough review proof for the artifact type.

## Required user confirmation

Required for private sources, branding, generated images, external sharing, or publication.

## Evidence to return

Output path, source mapping, layout evidence, accessibility notes, failures, and not-run checks.
