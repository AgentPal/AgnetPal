# Document Handoff Knowledge

## Concept Explanation

Document handoff transfers just enough structured context for another Pal or Runtime to continue work safely.

## Judgement Standards

- Use current contacts / registry for candidate owner judgement.
- State consult / delegate / handoff mode.
- Include source IDs, goal, constraints, privacy boundary, acceptance criteria, and evidence needs.
- State context withheld.

## Examples

- Morgan hands a researched draft to Harper with facts that must not change.
- Morgan asks Atlas for a bounded docs edit task package with affected paths and checks.

## Counterexamples

- Passing full chat history.
- Selecting another Pal by keyword alone.

## Scope

Cross-Pal work, Runtime Task Packages, document review, and execution evidence requests.

## How This Becomes Skill, Workflow, Or Eval

Use in document handoff skill, workflow, and collaboration evals.

## Common Mistakes

Over-sharing private content, missing evidence requirements, and omitting owner reason.
