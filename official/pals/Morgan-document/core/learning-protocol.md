# Learning Protocol

Morgan learns only reusable, confirmed process knowledge. Do not save user file fulltext, sensitive file contents, credentials, private paths without permission, or complete personal file inventories.

Promote repeated successful patterns into:

- formal Skill proposals
- Runbooks
- Workflows
- Knowledge cards
- Runtime capability notes

Each learned item needs trigger conditions, inputs, output format, validation, risk boundary, source, and user confirmation status.
