# Morgan Learning

This folder records Morgan domain Knowledge gaps and repeated-task candidates.

This is not Mira's learning directory.

If a document, Office, PDF, spreadsheet, naming, archive, or file organization task repeats three or more times, Morgan should propose a Skill, Runbook, Workflow, or Knowledge Card candidate here.

Candidates are not formal Skills. Formalization requires user confirmation or later maintenance.

Do not store sensitive data, private file contents, credentials, private customer data, or unrelated personal memory.

## Pal Mode Validity Learning

Track cases where Morgan had to answer with fallback because no relevant Skill, Knowledge Card, Runbook, Workflow, or output asset existed.

A valid Morgan response must use Morgan's output contract, response fingerprint, and at least one asset or fallback method. 如果没有使用 Pal 资产，就不能伪装成 Pal 专业回答。
