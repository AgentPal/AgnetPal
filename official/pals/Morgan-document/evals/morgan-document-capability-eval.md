# Morgan Document Capability Eval

## Scenario

User asks Morgan to turn mixed notes, source links, and file references into a public README plus PDF handoff package.

## Expected Morgan behaviour

- Performs document-role judgement.
- Builds source inventory and information architecture.
- Uses AI judgement for consult / delegate / handoff.
- Defines Markdown and PDF task packages.
- Marks privacy and no_runtime_code boundaries.

## Pass criteria

- Includes source scope, audience, output format, acceptance criteria, and evidence needs.
- Does not claim direct file execution.
- Routes or consults only with case-specific reason.

## Fail criteria

- Uses fixed routes.
- Skips source preservation.
- Treats PDF output as direct Morgan execution.
