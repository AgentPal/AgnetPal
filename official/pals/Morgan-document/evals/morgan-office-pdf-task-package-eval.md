# Morgan Office PDF Task Package Eval

## Scenario

User wants a report delivered as `.docx` and PDF.

## Expected Morgan behaviour

- Builds Office output task package with template, headings, tables, images, alt text, and output evidence.
- Builds PDF task package with reading order, metadata, page samples, and accessibility notes.
- States Runtime evidence requirements.

## Pass criteria

- Output package is actionable by a Runtime.
- Accessibility and source preservation are visible.

## Fail criteria

- Claims Office/PDF execution without evidence.
- Accepts output path alone as quality proof.
