# Morgan Skills

Morgan has legacy file/office skill folders plus 13 Document / File Workflow Lead skill cards.

## Document workflow skill cards

| Skill | Use |
| --- | --- |
| `document-intake-skill.md` | Clarify document request, audience, sources, format, privacy, and evidence. |
| `information-architecture-skill.md` | Design document structure, navigation, and doc-type classification. |
| `source-material-organization-skill.md` | Inventory and group source material before transformation. |
| `content-preservation-skill.md` | Preserve source details, claims, examples, and traceability. |
| `markdown-documentation-skill.md` | Plan Markdown docs using predictable headings and syntax. |
| `office-output-task-package-skill.md` | Prepare Word/PowerPoint/Excel output packages for a Runtime. |
| `pdf-output-task-package-skill.md` | Prepare PDF export/review packages with reading order and accessibility needs. |
| `file-workflow-design-skill.md` | Design safe document/file workflows without direct file operations. |
| `document-quality-review-skill.md` | Review completeness, consistency, accessibility, and source preservation. |
| `versioned-documentation-skill.md` | Plan docs-as-code and versioned documentation updates. |
| `release-notes-documentation-skill.md` | Prepare release notes and changelog content. |
| `document-handoff-skill.md` | Create document/file Context Packets and Runtime Task Packages. |
| `privacy-sensitive-document-review-skill.md` | Review privacy-sensitive document handling and minimization. |

All skills keep the `no_runtime_code` boundary.
