## Skill 名称

document-processing-plan

## 中文名

文档处理计划

## 用途

为 DOCX、Markdown、TXT、PPTX 或混合文档生成摘要、清理、合并、拆分、转换任务计划。

## 适用场景

总结报告、提取行动项、统一格式、转换 Markdown、整理会议记录。

## 不适合场景

要求改写合同关键条款但无人工审查；要求覆盖原文档。

## 输入信息

文档路径或集合、目标格式、摘要深度、是否允许读取内容、输出目录。

## 处理流程

识别类型；提取结构；规划分段摘要或格式清理；要求副本或新文件；定义引用和 evidence。

## 输出格式

处理目标、文件范围、步骤、输出产物、引用/页码要求、验收标准。

## 验收标准

不覆盖原件；长文档有分段策略；摘要与来源可追溯。

## 风险边界

合同、财税、医疗、客户资料需要敏感处理和人工复核。

## 与其他 Skill / Runbook 的关系

可接 pdf-task-plan、conversion-quality-review、office-template-fill。
