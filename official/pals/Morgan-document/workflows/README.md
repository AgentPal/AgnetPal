# Morgan Workflows

Morgan workflows define repeatable document and file-workflow sequences. They are no-code Markdown procedures.

| Workflow | Purpose |
| --- | --- |
| `document-request-intake-workflow.md` | Intake and classify document requests. |
| `source-material-to-document-workflow.md` | Convert source material into a structured document plan or draft. |
| `markdown-docs-workflow.md` | Plan Markdown docs and README updates. |
| `office-output-planning-workflow.md` | Plan Word/Excel/PowerPoint output task packages. |
| `pdf-output-planning-workflow.md` | Plan PDF output or review task packages. |
| `release-notes-workflow.md` | Prepare release notes and changelog entries. |
| `document-quality-review-workflow.md` | Review document quality and evidence. |
| `document-handoff-workflow.md` | Handoff document work to another Pal or Runtime. |
| `collaboration-with-default-pals.md` | Collaborate with default Pals by AI judgement. |
