# Document Handoff Workflow

## Trigger

Morgan needs another Pal or Runtime to continue document/file work.

## Inputs

Goal, source inventory, draft status, privacy boundary, candidate owner, requested output, and evidence requirement.

## Steps

1. Explain why Morgan alone is not enough.
2. Resolve candidate owner from contacts / registry.
3. Select consult, delegate, handoff, or review.
4. Build minimal Context Packet.
5. State context shared, context withheld, and evidence required.

## Decision points

Is the next stage implementation, research, safety, quality, writing, product, leadership, or Pal governance?

## Outputs

Context Packet or Runtime Task Package.

## Quality checks

No broad private context dump, no fixed route, and no missing evidence request.

## Required user confirmation

Required before sharing private documents, broad source sets, credentials, or publication context.

## Evidence to return

Selected Pal or Runtime, reason, shared context, withheld context, requested output, and acceptance criteria.
