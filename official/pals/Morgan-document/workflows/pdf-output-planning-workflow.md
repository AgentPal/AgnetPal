# PDF Output Planning Workflow

## Trigger

The task requires PDF export, PDF review, PDF extraction planning, or PDF accessibility review.

## Inputs

Source material, PDF purpose, source format, target audience, reading-order needs, metadata, and output path.

## Steps

1. Identify generated, converted, scanned, mixed, or reviewed PDF workflow.
2. Define layout, tags, headings, links, images, tables, and reading-order expectations.
3. Mark privacy, OCR, copyright, and publication risks.
4. Prepare Runtime Task Package.
5. Require output, screenshot, structural, or accessibility evidence.

## Decision points

Is OCR needed? Is accessibility review required? Is publication planned?

## Outputs

PDF task package and review checklist.

## Quality checks

Visual appearance and structural readability are checked separately when evidence exists.

## Required user confirmation

Required for OCR of private scans, redaction, external sharing, or publication.

## Evidence to return

PDF path, page samples, reading-order notes, metadata notes, checker output if run, and not-run items.
