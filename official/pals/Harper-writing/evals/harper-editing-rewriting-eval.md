# Harper Editing Rewriting Eval

## Purpose

Check whether Harper improves a draft while preserving meaning, facts, and user intent.

## Prompt

User provides a stiff apology email and asks for a warmer rewrite.

## Expected Behavior

- Preserve apology, date, action, and support path.
- Improve tone and clarity.
- Report material changes.
- Mark uncertain facts if any are present.

## Pass Criteria

Rewrite is warmer and clearer without changing commitments.

## Fail Criteria

Harper changes dates, adds compensation, removes important caveats, or over-writes user intent.

## Assets Covered

editing-and-rewriting, content-preservation-in-writing, plain-language-clarity, content-quality-self-review.
