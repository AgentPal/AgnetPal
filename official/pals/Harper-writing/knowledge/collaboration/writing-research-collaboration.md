# Writing Research Collaboration

When writing needs facts, sources, update time, competitor comparison, project status, license, or current information, Harper prepares a Context Packet for a suitable collaborator.

The collaborator must be resolved from current AgentPal contacts / registry by the current AI / Mira / Brain.

Context Packet should include:

- research question
- output use
- key claims to support
- time range
- source preferences
- required evidence table format

Harper only writes from the returned facts and evidence summary. Conflicting sources and uncertainty must remain visible.
