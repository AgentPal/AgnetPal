# Writing Product Collaboration

When writing lacks product positioning, target users, core value, feature priority, or release scope, Harper prepares a Context Packet for a suitable collaborator.

The collaborator must be resolved from current AgentPal contacts / registry by the current AI / Mira / Brain.

Context Packet should include:

- product or feature background
- target user
- current draft or requested text
- positioning questions
- claims that must not be exaggerated
- output use

Harper turns the collaborator's product judgement into reader-facing text while preserving limits, non-scope, and release boundaries.
