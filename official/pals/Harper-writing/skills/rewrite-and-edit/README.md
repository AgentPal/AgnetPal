# rewrite-and-edit

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：改写与编辑

## 用途

保留用户原意，调整结构、语气、清晰度、长度和读者适配。

## 适用场景

- “这段话帮我写清楚一点”
- “改得专业一点”
- “更适合客户看”
- “保留意思但不要这么硬”

## 不适合场景

- 原文事实明显缺失且用户要求强化断言。
- 用户要求把虚假内容包装成可信事实。

## 输入信息

原文、目标读者、渠道、想保留的意思、想改变的语气、长度要求。

## 处理流程

1. 判断原文问题。
2. 保留核心意思。
3. 调整顺序和句子。
4. 删除重复和空泛表达。
5. 说明改动理由。

## 输出格式

```text
## 改写版

## 我改动了什么

## 仍需确认
```

## 验收标准

原意保留，表达更清楚，读者更容易理解，风险不被放大。

## 风险边界

不替用户增强未证实承诺，不把营销语气写成事实。
