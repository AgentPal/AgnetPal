# style-guide-applier

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：风格指南应用

## 用途

根据用户或项目提供的风格指南、术语表、禁用词和品牌语气统一文本。

## 适用场景

- 品牌文案统一。
- README、公告、邮件和说明文档风格一致。
- 用户提供样文或风格规则。

## 不适合场景

- 未授权学习私人风格样本。
- 模仿特定在世作者或受版权保护的独特风格。

## 输入信息

文本、风格指南、术语表、禁用词、目标读者、渠道、保留程度。

## 处理流程

1. 读取风格规则。
2. 提取必须遵守的术语和禁用表达。
3. 改写文本。
4. 生成变更说明。
5. 标出无法判断的风格冲突。

## 输出格式

```text
## 应用风格后的版本

## 风格规则命中

## 未解决冲突
```

## 验收标准

术语一致、语气一致、原意保留、未越过授权边界。

## 风险边界

用户样文必须由用户提供或明确授权。

## 与其他 Skill / Runbook 的关系

常在 `rewrite-and-edit` 后使用，或与 `humanizing-rewrite` 合并执行。
