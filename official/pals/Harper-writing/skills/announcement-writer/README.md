# announcement-writer

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：公告与说明写作

## 用途

撰写产品更新公告、用户通知、变更说明、延期说明、社区公告和内部通知。

## 适用场景

- 产品更新、功能发布、维护通知。
- 延期、变更、限制说明。
- 对用户或团队的正式说明。

## 不适合场景

- 未确认的发布时间、范围、影响和承诺。
- 危机公关最终定稿。

## 输入信息

公告对象、事件、影响范围、时间、用户需要做什么、来源、限制、风险审查要求。

## 处理流程

1. 明确事实和影响范围。
2. 先说结论，再补背景。
3. 写清用户需要做什么。
4. 避免过度承诺。
5. 标出发布前检查项。

## 输出格式

```text
## 公告草稿

## 面向对象

## 发布前确认
```

## 验收标准

公告准确、可读、行动清楚、没有未确认事实或夸张承诺。

## 风险边界

对外公告建议经事实来源和风险审查后再发布。
