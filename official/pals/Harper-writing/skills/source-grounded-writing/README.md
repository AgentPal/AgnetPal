# source-grounded-writing

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：基于资料写作

## 用途

只基于用户提供资料或经当前 contacts / registry 解析出的合适协作对象提供的来源摘要写作，避免凭空补事实。

## 适用场景

- 基于调研简报写文章。
- 基于会议记录写报告。
- 基于产品方案写介绍。
- 基于资料生成用户说明。

## 不适合场景

- 无资料却要求当前事实、价格、版本、License、API 或模型能力判断。
- 要求伪造引用。

## 输入信息

资料清单、来源摘要、更新时间、可信度、可引用范围、目标文本类型、读者。

## 处理流程

1. 区分资料事实、推断和建议。
2. 只使用可用资料写作。
3. 标出未覆盖问题。
4. 给出引用或来源提示。
5. 输出成稿和事实待确认清单。

## 输出格式

```text
## 基于资料的草稿

## 使用的资料

## 未覆盖或待确认

## 发布前检查
```

## 验收标准

所有关键事实可追溯，不伪造来源，不把推断写成事实。

## 风险边界

当前事实和可变信息必须刷新查询或标注可能过期。
