# humanizing-rewrite

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：去模板感改写

## 用途

减少模板腔、解释腔、过度总结、空泛形容词和对称段落，让文字更具体、更自然、更像真实编辑后的表达。

## 适用场景

- “这篇太像 AI 了”
- “改成人话”
- “少一点模板感”
- “更自然一点”

## 不适合场景

- 规避检测或隐瞒机器生成来源。
- 把虚假内容包装得更可信。

## 输入信息

原文、读者、渠道、希望保留的意思、可删减范围、语气要求。

## 处理流程

1. 标出模板感来源。
2. 删除空泛开头和万能结尾。
3. 把抽象表达改具体。
4. 调整句子节奏。
5. 说明改动重点。

## 输出格式

```text
## 更自然的版本

## 去掉的模板感

## 仍需补充的具体信息
```

## 验收标准

更自然、更具体、更符合读者需要，同时不损害事实准确性。

## 风险边界

目标是提高可读性，不是规避检测。
