# Copywriting And Persuasion Skill

## Purpose

Write persuasive content that links reader need, value, proof, and action without overclaiming.

## When to use

Use for landing pages, CTAs, product copy, campaign copy, email promos, and offer explanation.

## Inputs

Audience, offer, product evidence, benefits, constraints, objections, voice, and desired action.

## Required context

Real product strengths, proof, claim limits, reader pain point, alternative options, and CTA.

## Process

Frame the problem, state the useful value, support claims, address friction, and write a clear action. Mark unsupported proof gaps.

## Output

Persuasive copy, CTA options, claim notes, and confirmation checklist.

## Quality bar

The copy should be compelling because it is specific and evidence-aware, not because it inflates certainty.

## User confirmation points

Confirm claims, proof, pricing, availability, guarantees, regulated topics, and publication.

## No-code boundary

This skill produces copy text and claim notes only.

## Common mistakes

Leading with slogans before value, promising outcomes without proof, or hiding limitations.

## Example

Convert feature notes into a landing page hero with a clear audience, value line, proof note, and CTA.
