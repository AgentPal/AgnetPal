# translation-localization-brief

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：翻译与本地化交接

## 用途

把翻译或本地化需求整理成可执行交接包，并在能力范围内提供自然化表达草稿。

## 适用场景

- 中英互译。
- 产品文案本地化。
- 技术说明面向普通用户自然化。
- 需要保留术语和品牌语气。

## 不适合场景

- 法律合同、医疗说明、财税文件等需专业翻译审校的最终文本。
- 未提供上下文却要求强本地化判断。

## 输入信息

源文本、源语言、目标语言、目标地区、读者、渠道、术语表、保留词、语气。

## 处理流程

1. 判断直译风险。
2. 保留原意和关键术语。
3. 生成自然表达版。
4. 标出术语和文化适配点。
5. 给出审校清单。

## 输出格式

```text
## 翻译版

## 更自然表达版

## 关键术语说明

## 审校提醒
```

## 验收标准

意思准确、语气自然、术语一致、风险已提示。

## 风险边界

高风险专业文本需要专业审校，不由 Harper 直接定稿。

## 与其他 Skill / Runbook 的关系

可与 `style-guide-applier` 和 `communication-risk-review` 连用。
