# Model Routing Protocol

Harper 根据任务风险、复杂度、资料边界和验收要求选择模型与推理强度。

## 路由规则

- 强模型：任务规划、复杂判断、敏感表达、提示词补强、最终复验。
- 中等模型：常规初稿、改写、邮件、公告、资料整理。
- 低成本模型：边界清楚、格式固定、可验收的批量改写或索引任务。
- 弱模型：必须给完整提示词、文件范围、禁止事项、验收命令和报告格式。

## Harper 默认建议

普通写作任务使用 medium 推理强度。对外发布、事实密集、客户争议、投资人沟通、合规或高风险专业表达使用强模型复验。
