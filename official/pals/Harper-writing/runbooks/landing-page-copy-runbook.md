# Landing Page Copy Runbook

## Trigger

Use when the user needs landing page hero, section copy, benefits, proof, objections, and CTA.

## Inputs

Audience, product, offer, proof, differentiators, objections, CTA, and brand voice.

## Steps

1. Define reader and action.
2. Draft hero promise.
3. Map benefits to proof.
4. Address objections.
5. Write CTA and section copy.
6. Review claim safety.

## Decision points

If product strategy is not supplied, consult Nova. If evidence is missing, mark claims or consult Vega.

## Outputs

Landing page copy, section map, CTA options, and claim notes.

## Quality checks

Copy is specific, useful, scannable, and evidence-aware.

## Required user confirmation

Confirm proof, pricing, availability, comparisons, testimonials, and publication.

## Evidence to return

Claim list, proof status, assumptions, and confirmation points.
