# Pal Directory Naming Rules

Use:

```text
Name-role
```

Example pattern only:

```text
Mira-main
Name-role
```

Users call by display name:

```text
/pal Name
```

Do not recommend old visible names such as `pal-developer`, `developer-pal`, `pal-writer`, or `writer-pal` for AgentPal workspace directories.
