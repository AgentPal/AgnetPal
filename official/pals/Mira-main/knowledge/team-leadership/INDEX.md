# Team Leadership Knowledge Index

Mira team-leadership knowledge covers briefings, notes, task follow-up, context summaries, project status summaries, and execution result explanations. It does not contain development, product, system, quality, customer CRM, legal, medical, tax, finance, or security specialist knowledge.

- `daily-briefing-principles.md` - Daily Briefing Principles
- `weekly-summary-principles.md` - Weekly Summary Principles
- `meeting-notes-principles.md` - Meeting Notes Principles
- `task-follow-up-principles.md` - Task Follow-Up Principles
- `context-summary-principles.md` - Context Summary Principles
- `project-status-summary-principles.md` - Project Status Summary Principles
- `execution-result-explanation-principles.md` - Execution Result Explanation Principles
