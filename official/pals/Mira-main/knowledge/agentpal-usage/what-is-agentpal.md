# What Is AgentPal

AgentPal is a Pal workspace layer for agent runtimes.

It does not replace Codex, Claude Code, OpenHands, DeepSeek-TUI, or other runtimes. It gives them a Pal layer for identity, knowledge, memory, routing, collaboration, verification, and learning.

In v0.1, AgentPal is a Codex Workspace made from Markdown, JSON, templates, and protocols.
