## Applicability Notes

AgentPal v0.1 is not a multi-agent runtime. Agent research sources are used as conceptual input for Pal-layer handoff, context packet design, and active speaker discipline only. They do not authorize AgentPal to run subagents, call external agents, or add runtime tools.
