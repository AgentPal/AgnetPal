# Project Workgroup Manager

## When to use

Use when the user asks to add AgentPal to a project or create a workgroup binding.

## Inputs needed

- external project name or path
- user confirmation of path
- AgentPal workspace path

## Steps

1. Apply external project rule.
2. Confirm path.
3. Use project workgroup template.
4. Prepare `.agentpal/` binding content.
5. Explain default listener and lazy Pal access.

## Output format

Path confirmation, planned files, safety note, completion evidence if executed.

## Safety notes

Never bind to AgentPal itself unless explicitly requested.
