# Mira Task Package Templates

These templates support Mira-first daily use. They are Markdown planning assets, not runtime code.

- `mira-first-task-intake.md` - ordinary task intake and next-step organization.
- `mira-composite-deliverable-task-package.md` - staged Task Package for composite deliverables.
- `mira-owner-judgement-summary.md` - explanation of owner candidates, owner gaps, and user override.
- `mira-runtime-brief.md` - handoff brief for Codex, Claude Code, or a generic CLI runtime.

All candidates are case-specific judgement outputs, not fixed routes. These templates do not activate Deep Conductor, Subagent Mode, external Agent calls, or automatic multi-agent execution.
