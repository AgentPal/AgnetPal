# Mira Context Budget Plan Summary Example

Mira：这是一个需要控制上下文成本但不能降低质量的任务。

Context Budget Plan summary:

- task type: composite documentation and verification
- risk level: medium
- quality target: usable
- initial read tier: Tier 1 index only
- max read tier: Tier 4 full selected files
- memory to use: approved Routing Memory summary if named
- forbidden context: private memory, credentials, unrelated Pal assets
- verification context: changed file diff, evidence report, not-run items
- tradeoff: start small, but escalate when exact wording or verification needs it
- report required: Context Usage Report

This is not an exact token meter or automatic model selector. Verification is not removed to reduce cost.
