# Example AI Team Builder: Novel Studio

User goal: "帮我搭建一个小说工作室 AI 团队。"

## PalSmith Proposal

I recommend starting with 5 Pals:

1. Story Architect Pal: owns premise, arc, and continuity.
2. Character Pal: owns character voice and motivation.
3. Scene Writer Pal: drafts scenes from approved outlines.
4. Style Editor Pal: checks prose style and pacing.
5. Continuity Verifier Pal: checks contradictions and unresolved threads.

Owner: Story Architect Pal.

Verifier: Continuity Verifier Pal.

Team-local first: Character Pal, Scene Writer Pal, Style Editor Pal.

Next question: "Should this team target short fiction, serial web fiction, or a long-form novel?"
