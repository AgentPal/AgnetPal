# Website Operator AI Team Blueprint

This is a blueprint, not an installed Pal. After user confirmation, PalSmith can generate corresponding Pal Creation / Team Creation Task Packages.

## 适合谁

Small website owners who need help with content planning, SEO basics, conversion copy, publishing checks, and analytics interpretation.

## 用户可以怎么说

```text
/pal PalSmith 帮我搭建一个网站运营 AI 团队。
```

## 推荐团队规模

5 Pals. This stays within the default 3-5 range.

## 推荐 Pal 列表

| Pal | Role | Responsibilities | Not Responsible For | Visibility |
| --- | --- | --- | --- | --- |
| Content Planner | content-strategy | topic map, publishing calendar, content briefs | final brand strategy | team-local |
| SEO Reviewer | seo | search intent notes, metadata suggestions, internal-link ideas | guaranteed ranking claims | team-local |
| Conversion Writer | copy | landing copy, CTA variants, FAQ drafts | pricing or legal claims | team-local |
| Publishing Checker | release-review | pre-publish checklist, broken-link notes, accessibility reminders | deploying the site | global candidate |
| Metrics Interpreter | analysis | plain-language metric summaries, experiment questions | automatic tracking or data scraping | team-local |

## Owner / Verifier / Consultant

- Owner: Content Planner.
- Verifier: Publishing Checker.
- Consultants: SEO Reviewer, Conversion Writer, Metrics Interpreter.

## Team-Local vs Global Contacts

Publishing Checker may be global because it reviews general quality. Keep SEO and metrics Pals team-local unless they have clearly scoped source access.

## Shared Knowledge

- audience profile
- brand voice
- content pillars
- site structure
- public analytics summaries supplied by user
- legal claim boundaries

## Shared Workflows

- content brief creation
- SEO review
- landing copy pass
- pre-publish quality check
- monthly metrics review

## Context Packet Rules

Pass only the page, brief, keyword set, or metrics extract needed for the task. Do not assume access to private analytics or CMS credentials.

## Quality Gate

Before stable status, the team must reject guaranteed ranking claims and must not claim it deployed, scanned, or crawled the site without Runtime evidence.

## Eval Lab Starter

- Create a content brief.
- Review a page title and description.
- Draft conversion copy alternatives.
- Check a pre-publish page checklist.
- Interpret user-provided metrics.

## Runtime Task Package 入口

Start with `AI Team Builder Task Package`. Use `Pal Publish Quality Gate Task Package` before any public website-facing claims.

## 用户确认问题

- What kind of website is this?
- Which pages or content pillars matter first?
- Are analytics summaries allowed to be read?
- Should any Pal become a global contact?
- Is the first output a plan, a team draft, or file creation?

## 不建议一开始创建的 Pal

- CMS automation Pal.
- Web crawler Pal.
- Ad-spend automation Pal.
- Legal compliance Pal without approved sources.
