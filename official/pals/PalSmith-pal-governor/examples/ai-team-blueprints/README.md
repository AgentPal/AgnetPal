# PalSmith AI Team Blueprints

Status: PalSmith v0.4 example blueprints.

These Markdown files are blueprint examples only. They are not installed Pals, not official bundled Pals, and not registry / contacts entries. After user confirmation, PalSmith may turn a blueprint into Pal Creation or Pal Team Creation Runtime Task Packages.

## Blueprints

- `cross-border-commerce-team-blueprint.md`: cross-border commerce operations.
- `law-firm-team-blueprint.md`: small law firm knowledge and client-work support.
- `novel-studio-team-blueprint.md`: fiction writing studio.
- `website-operator-team-blueprint.md`: website operations and content growth.
- `product-development-team-blueprint.md`: product discovery and development support.

## Scale Rule

Default to 3-5 Pals. More than 5 Pals requires a reason. More than 8 Pals requires explicit second confirmation from the user.

## Boundary

Blueprints do not create files by themselves. PalSmith must still ask confirmation and prepare the correct Runtime Task Package before the current Agent Runtime writes anything.
