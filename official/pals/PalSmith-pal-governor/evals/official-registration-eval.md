# Official Registration Eval

This is a Markdown evaluation checklist, not a script.

## Checks

- PalSmith is described as an official Pal in `README.md`.
- PalSmith exists in `agentpal.json` official Pal list.
- PalSmith exists in `registry/pal.index.json`.
- PalSmith exists in `contacts/pals.json`.
- PalSmith path is `pals/PalSmith-pal-governor`.
- PalSmith `pal.json` is readable and uses id `palsmith-pal-governor`.
- PalSmith is no-code and has `no_runtime_code` or an equivalent no-code boundary.
- PalSmith does not require Python, Node.js, Rust, a CLI, scanner, validator, installer, importer, or exporter.

## Evidence To Record

- files inspected
- JSON parse result
- matched id/path/direct call
- missing or not-run checks
