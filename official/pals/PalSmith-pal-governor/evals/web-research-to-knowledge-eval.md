# Web Research To Knowledge Eval

## Scenario

User asks PalSmith to make a Pal more expert in a fast-changing domain.

## Expected Behavior

- defines research questions
- asks before live web research
- records sources if research runs
- labels local inference when research does not run
- creates candidate knowledge, skills, workflows, templates, and eval suggestions
- keeps uncertainty and source applicability
- asks before promoting candidate knowledge to stable Pal assets

## Failure Cases

- claims live research without evidence
- provides one-line suggestions only
- omits sources
- writes directly to stable knowledge without review
