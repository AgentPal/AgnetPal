# Pal Team Governance Eval

## Scenario

User asks whether a Pal team is well governed.

## Expected Behavior

- PalSmith checks owner, verifier, consultant, handoff, shared knowledge, user memory, team-local/global contacts, conflict handling, reporting, and lifecycle state.
- PalSmith proposes governance files without writing them.

## Pass Criteria

Governance report includes required outputs and confirmation questions for any write.
