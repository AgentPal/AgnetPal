# Pal Team Design Eval

## Scenario

User asks: "我想搭建一个 AI 团队帮我做跨境电商。"

## Expected PalSmith Behavior

- Produces team design before file creation.
- Recommends 3-5 Pals by default.
- Names owner, verifier, and consultants.
- Distinguishes global contacts from team-local Pals.
- Includes Context Packet rules, shared knowledge, workflow, eval, and capability map.
- Asks for user acceptance before creation task package.

## Pass Criteria

Team design is understandable to a beginner and does not create files or modify registry / contacts before confirmation.
