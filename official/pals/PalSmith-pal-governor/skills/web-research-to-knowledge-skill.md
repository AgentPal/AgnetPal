# Web Research To Knowledge Skill

## Purpose

Turn Runtime-performed web research into sourced candidate knowledge, skills, workflows, templates, and risk boundaries for a Pal.

## When To Use

Use when a Pal role needs current industry practices, role expertise, tools, output formats, regulatory boundaries, or public reference material.

## Inputs

- target Pal role and responsibilities
- research questions
- allowed web/search scope
- desired recency or source type
- write target for candidate knowledge

## Required Context

Read web-research-to-knowledge protocol, job expertise research knowledge, content preservation principles, and target Pal design.

## Process

1. Define research questions from the Pal role.
2. Ask user confirmation if network access or external data is involved.
3. Current Runtime performs search/read actions and records sources.
4. Assess source credibility, date, applicability, and uncertainty.
5. Extract job knowledge, skill requirements, common workflows, output templates, and risks.
6. Produce candidate knowledge and candidate skills.
7. Keep source citations and scope notes.
8. Ask user confirmation before writing stable Pal knowledge.

## Output

- research task package
- source inventory
- credibility notes
- candidate knowledge
- candidate skill/workflow/template/eval suggestions
- uncertainty list

## Quality Bar

Research output must be specific enough to improve the Pal's job competence and must distinguish live research from local inference.

## User Confirmation Points

- network research
- source scope
- writing candidate knowledge
- promoting candidate knowledge to stable Pal knowledge

## No-Code Boundary

PalSmith does not include a crawler or downloader. The current Agent Runtime performs web access and returns evidence.

## Common Mistakes

- pretending research ran
- writing only a one-line recommendation
- omitting source date and applicability
- promoting uncertain research directly to stable knowledge

## Example

For a website operator Pal, Runtime research can collect current search intent practices, content quality guidance, accessibility checks, and analytics interpretation sources; PalSmith then turns them into candidate knowledge and workflows with source notes.
