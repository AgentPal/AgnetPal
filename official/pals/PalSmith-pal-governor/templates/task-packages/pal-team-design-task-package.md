# Pal Team Design Task Package

task_name: Pal Team Design Task Package
owner_pal: PalSmith
executing_runtime: current Agent Runtime

## User Goal

Design an AI team from a user's goal before creating files.

## Required Design Fields

- user goal
- business scenario
- recommended Pal count
- proposed Pal names
- roles and responsibilities
- owner Pal
- verifier Pal
- consultant Pals
- Context Packet rules
- global contacts candidates
- team-local candidates
- shared knowledge
- shared workflows
- team eval
- team capability map

## Confirmation Questions

- Do you accept this team design?
- Should any member use an existing Pal?
- Which Pals should be global contacts?
- May the Runtime create the team files after the creation task package is generated?

## Expected Output

Team design proposal only. Creation requires a later Pal Team Creation Task Package.
