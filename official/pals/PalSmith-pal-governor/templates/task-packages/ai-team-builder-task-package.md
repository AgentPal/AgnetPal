# AI Team Builder Task Package

task_name: AI Team Builder Task Package
owner_pal: PalSmith
executing_runtime: current Agent Runtime

## User Goal

Design a Pal team from a plain-language user goal.

## Required Fields

- user goal understanding
- scenario judgement
- recommended team size
- proposed Pal list
- each Pal's responsibilities
- each Pal's non-responsibilities
- owner Pal
- verifier Pal
- consultant Pals
- team-local vs global-contact candidates
- shared knowledge
- shared workflow
- team Context Packet
- team capability map draft
- team Eval Lab draft
- user confirmation questions

## Confirmation Questions

- Do you accept this team design?
- Should this stay small, or do you want a complex team?
- Which Pals should become global contacts?
- May the Runtime generate the team creation package after this design?

## Boundary

This task package designs the team. It does not create files, update registry, update contacts, or read private memory.
