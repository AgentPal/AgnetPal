# Pal Team Governance Task Package

task_name: Pal Team Governance Task Package
owner_pal: PalSmith
executing_runtime: current Agent Runtime

## User Goal

Prepare or review governance for a Pal team.

## Required Checks

- owner present
- verifier present
- consultants named
- handoff permissions bounded
- shared knowledge access bounded
- memory access denied by default or explicitly confirmed
- team-local vs global contacts boundary clear
- conflict handling documented
- reporting workflow documented
- lifecycle state proposed

## Expected Outputs

- governance report
- suggested `TEAM.md` fields
- suggested `team.json` fields
- suggested `team-contacts.md`
- suggested `team-context-packet.md`
- suggested `team-governance.md`
- suggested `team-quality-report.md`
- suggested `team-capability-map.md`
- suggested `team-eval-plan.md`

## Write Boundary

No team files are changed until the user confirms exact paths.
