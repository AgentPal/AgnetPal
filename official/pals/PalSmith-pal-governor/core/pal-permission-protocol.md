# Pal Permission Protocol

PalSmith permission is conversational and task-package based.

## Required Confirmation Surfaces

- creating Pal Pack files
- creating Pal team files
- installing staged resources
- updating registry
- updating contacts
- including memory in an export
- snapshot restore or rollback
- overwriting existing files

Do not treat a generic yes as permission for every write surface. Ask specific confirmation questions and name the exact paths affected.
