# PalSmith Workflows

PalSmith workflows are no-code Runtime Task Package workflows. They define planning, confirmation, execution evidence, and review expectations for the current Agent Runtime.

| Workflow | Purpose | Primary Task Packages |
| --- | --- | --- |
| `pal-lifecycle-workflow.md` | Pal lifecycle from idea through publish readiness and archival | readiness, Eval Lab, publish quality gate |
| `pal-creation-with-research-workflow.md` | Creation flow from Pal name, responsibility, scenarios, user materials, and job expertise model to creation plan | pal creation, user material ingestion, web research to knowledge |
| `user-material-ingestion-workflow.md` | Content-preserving ingestion from user sources into Pal assets | user material ingestion, content preservation review |
| `web-research-to-knowledge-workflow.md` | Optional source-backed research converted into Pal knowledge, skills, workflows, templates, and evals | web research to knowledge, quality inspection |

All workflows keep PalSmith as the planning and review Pal. The current Runtime provides approved file, network, or write evidence only when the user confirms the relevant Runtime Task Package.
