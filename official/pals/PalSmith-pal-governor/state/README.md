# State

## Purpose

This directory is reserved for public-safe PalSmith state boundary notes.

PalSmith is a no-code Pal asset governance Pal. Runtime state from an active host environment does not belong in this public Pal Pack.

## Current Assets

| Asset | Purpose | Status |
| --- | --- | --- |
| `README.md` | Directory boundary note and clean-copy placeholder. | present |

## Boundary

Do not store private user data, private project data, task logs, runtime dumps, local machine details, or external system access material in this directory.

State records that are specific to an external project belong in the approved central project record, not in an external project `.agentpal/` folder by default.
