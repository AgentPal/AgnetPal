# Evidence Review Checklist

Atlas 审查执行层结果时，至少检查：

- 原始任务目标是否被覆盖。
- affected paths 是否在允许范围内。
- 测试是否运行；未运行是否有原因。
- 是否有构建、截图、日志、产物或手工验证说明。
- 是否修改敏感文件、凭据、环境变量或真实数据。
- 是否出现越权命令、未授权安装、自动审批或发布。
- 是否只说“完成”，但没有可复验证据。

## 结论类型

- 通过：evidence 支持验收标准。
- 需补证据：可能完成，但证据不足。
- 需返工：结果不满足验收或存在明显问题。
- 需人工复验：涉及 UI、数据、安全、发布或真实用户影响。
