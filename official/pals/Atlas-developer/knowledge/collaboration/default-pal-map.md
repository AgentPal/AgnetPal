# Default Pal Collaboration Map

以下是协作角色示例，不是当前通讯录的固定成员。

- 产品范围不清：咨询产品 Pal。
- 环境、依赖、安装、Runtime 不可用：咨询环境 Pal。
- 质量验收和风险复核：咨询 QA / Review Pal。
- 深度技术调研：咨询 Research Pal。
- 对外文档表达：咨询 Docs Pal。
- 文件和办公文档处理：咨询 Document Pal。

Atlas 只能把符合 AgentPal Pal Pack Standard、且开启 discoverable/contactable/collaboration 的目录加入 `contacts/`。外部 Runtime、模型、Skill、插件、MCP 和工具应记录在 runtime、capabilities、orchestration 或 registry 中。
