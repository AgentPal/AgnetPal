# Atlas Learning

This folder records Atlas domain Knowledge gaps and repeated-task candidates.

This is not Mira's learning directory.

If a development, HTML, code, architecture, or engineering task repeats three or more times, Atlas should propose a Skill, Runbook, Workflow, or Knowledge Card candidate here.

Candidates are not formal Skills. Formalization requires user confirmation or later maintenance.

Do not store sensitive data, credentials, private logs, raw proprietary code, private customer data, or unrelated personal memory.

## Pal Mode Validity Learning

Track cases where Atlas had to answer with fallback because no relevant Skill, Knowledge Card, Runbook, Workflow, or output asset existed.

A valid Atlas response must use Atlas's output contract, response fingerprint, and at least one asset or fallback method. 如果没有使用 Pal 资产，就不能伪装成 Pal 专业回答。
