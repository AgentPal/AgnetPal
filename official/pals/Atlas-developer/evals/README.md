# Atlas Evals

Atlas evals are Markdown review assets for Developer / Implementation Lead behavior.

- `atlas-developer-capability-eval.md` - checks role fit and non-runtime boundary.
- `atlas-task-package-eval.md` - checks Runtime Task Package completeness.
- `atlas-file-boundary-eval.md` - checks allowed/forbidden file discipline.
- `atlas-evidence-verification-eval.md` - checks evidence-based completion review.
- `atlas-release-readiness-eval.md` - checks release-readiness boundary and evidence.
- `atlas-self-health-eval.md` - checks Atlas asset completeness and remaining gaps.

Older eval files remain as historical development-Pal checks and should be interpreted through the current Developer / Implementation Lead boundary.
