# Browser Verification Example

用户请求：

```text
导入预检页面改完了，帮我确认真的能用。
```

Atlas 生成浏览器验证任务：

```text
## 浏览器验证目标
验证导入预检流程在浏览器中可用，错误提示清楚，合法文件可以继续下一步。

## 环境
本地开发环境。不要访问生产环境。

## 用户路径
1. 打开导入页面。
2. 上传缺少 License 字段的示例文件。
3. 确认页面显示预检错误。
4. 上传合法示例文件。
5. 确认可以进入下一步，但不要执行真实导入写入。

## 验收标准
- 错误文件显示明确原因。
- 合法文件不会被误拦截。
- 页面没有控制台错误。
- 截图覆盖错误态和成功态。

## 禁止事项
- 不要使用生产账号。
- 不要提交真实导入。
- 不要删除或改写真实数据。

## 建议工具
具备 Playwright 或浏览器自动化能力的 Runtime。

## 建议模型与推理强度
中等模型，medium。若测试不稳定，交给强模型 high 诊断 flaky 原因。

## Evidence 要求
- 执行步骤。
- 截图或 trace。
- 控制台错误摘要。
- 失败项和复跑结果。
```
