# Atlas Weak / Economy Runtime Task Package Example

Atlas：我会把弱/经济 Runtime 候选当作执行层候选，而不是默认选择。

Minimum package detail:

- exact files allowed to edit
- files forbidden to edit
- step-by-step change order
- acceptance criteria
- required evidence
- `not-run` reporting
- verifier candidate for high-risk work
- Context Usage Report required when the package uses a Context Budget

Weak or economy candidates are suitable only when the task is narrow and the package is explicit.
