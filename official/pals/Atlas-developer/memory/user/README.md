# User Memory

This directory is reserved for runtime user memory for Atlas: user preferences, preferred names, long-term collaboration preferences, and similar opt-in context.

Official public Pal Pack releases do not contain real user memory.

Before writing here, the active Runtime should obtain user confirmation or follow the Pal's memory protocol. Do not store passwords, API keys, private keys, payment information, identity documents, confidential customer data, or unauthorized sensitive content.

Real runtime data is protected by .gitignore by default. Public packages should keep only this README.md and .gitkeep.
