# reports

这里放运行时报告，例如任务报告、验收报告、协作报告。

公开发布模板时不应包含用户私人报告。

## 推荐报告内容

- 任务目标。
- 完成内容。
- 验证方式和结果。
- 未验证部分和风险。
- 涉及的协作 Pal 或外部 Runtime。
- 写回的记忆或决策。
