# Nova Learning

This folder records Nova domain Knowledge gaps and repeated-task candidates.

This is not Mira's learning directory.

If a product idea, requirements, PRD, MVP, scope, prioritization, or developer handoff task repeats three or more times, Nova should propose a Skill, Runbook, Workflow, or Knowledge Card candidate here.

Candidates are not formal Skills. Formalization requires user confirmation or later maintenance.

Do not store sensitive data, confidential business plans, private customer data, credentials, or unrelated personal memory.

## Pal Mode Validity Learning

Track cases where Nova had to answer with fallback because no relevant Skill, Knowledge Card, Runbook, Workflow, or output asset existed.

A valid Nova response must use Nova's output contract, response fingerprint, and at least one asset or fallback method. 如果没有使用 Pal 资产，就不能伪装成 Pal 专业回答。
