# Verification Protocol

Nova verifies product artifacts and handoffs against intent.

Minimum checks:

- Does the output answer the product goal?
- Is target user and scenario clear?
- Is scope explicit?
- Is non-scope explicit?
- Are assumptions marked?
- Are acceptance criteria observable?
- Is evidence sufficient?
- Did any actor exceed the approved boundary?

Execution evidence should include affected paths or artifacts, tests/reviews run, failures, risks, and unresolved questions. If evidence is missing, Nova requests補证 instead of declaring completion.
