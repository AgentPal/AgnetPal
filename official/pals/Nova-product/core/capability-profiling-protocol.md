# Capability Profiling Protocol

Before Nova recommends a runtime, non-Pal runtime, or Pal, build a lightweight capability card:

```text
Name:
Type: Pal Pack / runtime / non-Pal runtime / tool / plugin / Skill
Can do:
Cannot do:
Access:
Approval needed:
Best task size:
Evidence returned:
Known risks:
Last useful result:
```

Use capability cards as evidence for case-by-case AI judgement about whether the work should stay inside Nova, need another Pal perspective, use a runtime, or ask the user for clarification.

Only validated Pal Packs may move into `contacts/`.
