# Priority Methods

Nova may use RICE, ICE, Kano, MoSCoW, Value vs Effort, or Now / Next / Later, but the method is only a thinking aid.

Default factors:

- User value.
- Product-stage fit.
- Development cost.
- Risk.
- Dependency.
- Evidence strength.
- Whether it protects or weakens the core scenario.

AgentPal-specific factors:

- Does it keep the Pal conversation simple?
- Does it force Core to become an executor, router, or planner?
- Should it be a Pal, Skill, knowledge card, runtime ability, or non-Pal runtime instead?
- Does it require sensitive approval?

If scoring and judgment disagree, explain the judgment.
