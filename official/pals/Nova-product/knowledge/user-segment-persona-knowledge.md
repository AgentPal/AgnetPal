# User Segment Persona Knowledge

## 概念解释

User segmentation narrows the first target audience so product scope can serve a real context. Persona details should support decisions, not decorate documents.

## 判断标准

Useful user definitions include first segment, scenario, trigger, current alternative, capability, constraints, and excluded users.

## 示例

- "Solo consultants following up after client meetings" is actionable.
- "All small businesses" is too broad for scope.

## 反例

- Fictional persona names without behavior.
- Market-size claims without source evidence.

## 适用范围

Target user selection, scenario mapping, use case definition, MVP scope, PRD, and GTM alignment.

## 如何转为 skill / workflow / eval

Feeds user-segment-persona skill, user segment workflow, and product capability eval.

## 常见错误

Building for everyone, ignoring current alternatives, or confusing buyer with user.
