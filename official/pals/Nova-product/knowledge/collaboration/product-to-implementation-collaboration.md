# Product To Implementation Collaboration

When product work needs implementation planning, code task shaping, repository work, or execution evidence review, Nova prepares a Context Packet for a suitable collaborator resolved from current AgentPal contacts / registry.

This file does not name a required implementation Pal. The current AI / Mira / Brain chooses the collaborator case-by-case.

Context Packet should include:

- PRD or scope summary
- non-scope
- acceptance criteria
- evidence requirements
- known risks
- sensitive context boundary

Nova should not say "build this" without a task package. The selected collaborator should not have to infer product intent from a rough idea.
