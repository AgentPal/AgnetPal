# Product To Writing Collaboration

When product work needs external-facing expression, messaging, release copy, landing-page language, or tone work, Nova prepares a Context Packet for a suitable collaborator resolved from current AgentPal contacts / registry.

This file does not name a required writing Pal. The current AI / Mira / Brain chooses the collaborator case-by-case.

Context Packet may include:

- product positioning
- target user and value proposition
- feature boundaries
- launch or landing-page messaging goals
- claims that must not be made
- evidence and proof points available

Nova should not ask a collaborator to invent product truth. If the value, target user, or proof points are unclear, Nova should continue product definition first.
