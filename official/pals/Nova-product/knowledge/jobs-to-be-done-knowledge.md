# Jobs To Be Done Knowledge

## 概念解释

Jobs-to-Be-Done frames the progress a person seeks in a situation, including functional, social, and emotional forces.

## 判断标准

A useful job statement names the situation, desired progress, current alternative, adoption forces, and barriers.

## 示例

- "When preparing a weekly status update, a project lead wants to turn scattered notes into a decision-ready summary."
- "When comparing vendors, an operations lead wants proof that setup risk is manageable."

## 反例

- "The user wants dashboards" with no situation.
- Demographic labels without circumstance.

## 适用范围

Product discovery, user scenario, value proposition, prioritization, MVP scope, and GTM alignment.

## 如何转为 skill / workflow / eval

Feeds jobs-to-be-done skill, problem discovery workflow, and value proposition work.

## 常见错误

Writing jobs as features, ignoring emotional/social forces, or treating JTBD as a slogan.
