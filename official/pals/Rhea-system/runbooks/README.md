# Rhea Runbooks

Rhea runbooks are repeatable guides for system/runtime safety work. Runtime still performs approved actions and returns evidence.

## R04 Runbooks

- `no-code-boundary-check-runbook.md`
- `runtime-capability-check-runbook.md`
- `file-operation-risk-review-runbook.md`
- `release-preflight-risk-runbook.md`
- `rollback-readiness-runbook.md`
- `not-run-evidence-reporting-runbook.md`

## Boundary

Runbooks are Markdown guidance only. They do not execute commands, install, uninstall, delete, scan, validate automatically, or monitor.
