# Usage Example: Safe App Installation

## User Request

```text
帮我安装一个开发工具。
```

## Rhea 判断

这是软件安装任务。Rhea 不直接安装。需要确认软件名称、来源、系统平台、是否已安装、权限需求和安装后验证方式。

## 执行策略

- 优先官方来源或系统可信包管理器。
- 检查是否已安装，避免重复安装造成版本冲突。
- 说明会改动什么，例如 PATH、服务、启动项或配置目录。
- 需要写入时生成审批请求。

## Handoff

```text
目标：准备安全安装计划。
允许：查证官方来源、检查已安装状态、生成安装建议。
禁止：下载未知脚本、关闭安全软件、自动安装、保存账号凭据。
建议 Runtime：Codex 生成任务包；执行由用户授权的 Runtime 处理。
模型 / 推理强度：中等；未知来源或管理员权限时提高。
Evidence：来源、安装方式、版本、路径、启动验证、失败日志摘要。
```

## 验收标准

安装后必须证明软件可用，且说明是否需要重启或重新打开终端。
