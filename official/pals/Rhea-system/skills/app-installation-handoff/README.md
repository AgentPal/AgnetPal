# app-installation-handoff

运行时 Skill 入口请看 SKILL.md；README.md 仅作为人类阅读说明。


中文名：应用安装交接

## 用途

为软件安装生成安全、可审批、可验证的执行层任务包。

## 适用场景

- 用户要安装开发工具或普通应用。
- 需要选择 WinGet、Scoop、Homebrew、官方安装包等方式。
- 安装前需要说明影响范围。

## 不适合场景

- 破解软件、未知来源安装包、驱动自动安装或关闭安全软件。
- 用户要求 Rhea 直接安装。

## 输入信息

- 软件名称。
- 操作系统。
- 推荐来源或需要查证的来源。
- 是否允许包管理器、管理员权限或重启。

## 处理流程

1. 确认软件和来源。
2. 检查是否已安装和可能冲突。
3. 选择低风险安装方式。
4. 说明会影响什么。
5. 生成执行层安装任务和验证要求。

## 输出格式

```text
## 软件
## 推荐来源
## 当前状态
## 安装方式
## 可能改动
## 审批点
## 验证方式
```

## 验收标准

- 安装方式有来源或待查证标记。
- 风险和审批点清楚。
- 验证包括版本、路径、启动状态或程序列表。

## 风险边界

不直接下载、安装、升级或覆盖旧版本。

## 与其他 Skill / Runbook 的关系

使用 `runbooks/app-installation/safe-installation-handoff.md`，必要时调用 `permission-risk-review`。
