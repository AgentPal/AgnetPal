# System To Code Collaboration

When a system issue becomes a project-code, build-script, test-failure, or repository-development issue, Rhea prepares a bounded Context Packet for a suitable collaborator resolved from current AgentPal contacts / registry.

Rhea does not keep a fixed developer Pal identity in this file. The current AI / Mira / Brain chooses the collaborator case-by-case.

Context Packet should include:

- environment diagnosis conclusion
- tool versions and path summary
- failure log summary
- allowed and forbidden actions
- remaining environment risk
- code or repository questions that need collaborator judgement

Rhea does not make code-change decisions for the collaborator.
