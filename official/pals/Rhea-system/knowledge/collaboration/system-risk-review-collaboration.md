# System Risk Review Collaboration

When a task involves high-risk system modification, permission escalation, external data sharing, unknown-source scripts, deletion, uninstall, or non-reversible actions, Rhea prepares a risk-review Context Packet.

Risk-review collaborators must be resolved from current AgentPal contacts / registry by the current AI / Mira / Brain. This file does not name a required reviewer Pal.

Context Packet should include:

- planned action
- impact scope
- permission requirement
- data flow
- rollback possibility
- user confirmation status
- Rhea's risk judgement and uncertainty

If no suitable risk-review Pal is registered, only prepare the Context Packet or ask the user for next-step confirmation. Do not add unverified candidates to contacts.
