# Examples

Examples show expected AgentPal behavior using public-safe, synthetic scenarios.

They are documentation only. They do not claim commands were executed, files were changed, or external runtimes were called.

## Quick Start Examples

Start here for the three current user paths:

- `quickstart/codex.md`
- `quickstart/claude-code.md`
- `quickstart/generic-cli-agent.md`

## Main Example Areas

| Area | Purpose |
| --- | --- |
| `quickstart/` | Copyable examples for Codex, Claude Code, and Generic CLI Agent paths. |
| `runtime/` | Runtime guide examples and project-first notes. |
| `usage/` | Simple Pal Mode, direct Pal calls, handoff, Task Package, and project binding examples. |
| `pal-collaboration/` | Navigation for `/pal Mira`, `/pal Atlas`, and owner handoff examples. |
| `project-workgroup/` | External project binding examples. |
| `failures/` | Failure cases and corrected behavior. |
| `orchestration/` | Current/future orchestration methodology examples. Future examples do not activate runtime behavior. |

## Public-Safe Rules

- Use placeholders such as `<your-project>` and `<path-to-AgentPal>`.
- Do not include real user data, customer data, secrets, local absolute paths, or internal development logs.
- Do not make examples look like real project records.
- Do not describe Deep Conductor, Subagent Mode, parallel child workflows, or external Agent orchestration as active v0.1 behavior.
