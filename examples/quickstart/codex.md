# Quick Start: Codex

## User Flow

```text
Open the AgentPal Workspace in Codex.
Copy prompts/codex/initialize-agentpal-workspace.md.
Paste it into Codex.
```

## Expected Result

- Codex treats the current directory as the AgentPal Workspace.
- Mira becomes the default Main Pal / Leader / Conductor.
- The first welcome starts with `Mira：`.
- Ordinary messages go to Mira.
- Direct Pal calls use `/pal Name`.
- Current task handling remains `Simple Pal Mode only`.

## Not Expected

- no project binding is created
- no runtime dependency is installed
- no Subagent Mode is started
- no Deep Conductor workflow is executed
