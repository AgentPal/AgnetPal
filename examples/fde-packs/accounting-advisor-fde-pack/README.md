# Accounting Advisor FDE Pack

This is a public-safe example FDE / Expert Delivery Pack for an accounting advisor delivery method.

It demonstrates how reusable accounting advisory experience can be packaged without including real customer ledgers, tax filings, invoices, credentials, company names, financial privacy, or customer databases.

## Files

- `FDE.md`
- `fde.json`
- `reusable-assets.md`
- `customer-private-boundary.md`
- `verification-checklist.md`

## Boundary

This example is not accounting advice for any real customer. It is a public-safe structure example.

Human review by a qualified accounting or tax professional is required before using any accounting, tax, payroll, audit, or financial reporting output in a real engagement.
