# Single Entry Mira Example

## User

```text
I want to know whether the next version is worth doing and how to verify it.
```

## Expected AgentPal Shape

The user does not need to choose Nova, Atlas, Quinn, Codex, Claude Code, a Skill, or a future Deep Conductor path.

Mira should:

- receive the goal
- inspect current contacts / registry
- judge ownership case-by-case
- use Fast Route to one owner Pal if a clear owner exists
- prepare a Task Package or future-design plan if the task needs staged work
- explain uncertainty when no owner is clear

Explicit `/pal Name` still works when the user wants direct control.
