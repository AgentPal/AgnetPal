# Fast Route Example

This is a non-binding orchestration example.

## Situation

The user asks for a small, low-risk work product that one owner Pal can handle with a bounded context slice.

## Shape

```text
Mira -> owner Pal -> same-response answer or Task Package
```

## Context

- user goal
- contacts / registry
- selected owner Pal Level 0 assets
- one relevant index or asset if needed

## Why It Works

Fast Route avoids unnecessary workflow overhead. It still uses AI Judgement and output contracts.

## Not A Rule

This example does not create a fixed route for any task type.
