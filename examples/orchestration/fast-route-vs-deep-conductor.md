# Fast Route Vs Deep Conductor

## Fast Route In v0.1

User:

```text
Help me make a development plan for this project.
```

Shape:

```text
Mira -> one owner Pal -> answer
```

Mira gives a short ownership judgement and hands off. The owner Pal answers in Simple Pal Mode.

Fast Route is active in v0.1.

## Deep Conductor Future Design

User:

```text
Design a multi-runtime workflow to build, test, review, and release this project with independent verification.
```

Shape:

```text
Mira -> topology design -> Context Access Lists -> independent work/review -> synthesis -> routing memory
```

In v0.1, Mira may describe this as a Task Package or design plan. AgentPal does not execute Deep Conductor.
