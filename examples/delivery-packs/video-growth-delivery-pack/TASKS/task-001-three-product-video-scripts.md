# Task 001: Three Product Video Scripts

## Task type

Task Package example.

## Goal

Draft three public-safe short-video script variants for one product introduction video.

## Owner judgement input

Candidate owner: Scriptwriter Pal, with Research Pal and Brand Guardian Pal as review perspectives. This is not an automatic route.

## Inputs

- product: simulated product intro for a lightweight team planning tool;
- audience: small content teams;
- value proposition: fewer scattered notes before weekly publishing;
- proof boundary: no quantified performance claim;
- CTA: ask viewers to try a planning checklist.

## Steps

1. Summarize the audience pain and public-safe assumption.
2. Draft three 30-45 second scripts:
   - problem-solution;
   - demo-first;
   - objection-first.
3. Mark lines that need source material.
4. Mark unsupported claims as blocked.
5. Send to Brand Guardian Pal perspective for human-review-ready notes.

## Deliverables

- script variant A;
- script variant B;
- script variant C;
- material needs list;
- claim risk list;
- human review note.

## Acceptance

- exactly three variants are present;
- no real customer data is used;
- every claim is source-backed or marked blocked;
- final state is `requires-human-review`, not `pass`.

## Boundary

This task does not create videos, publish content, call an API, or write to an external project.
