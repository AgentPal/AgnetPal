# Video Growth Delivery Pack

This is a public-safe near-real runnable Delivery Pack example for AgentPal.

Business scenario: a content team wants a short-video growth AI team for product intro, founder talking, customer case, live clip, and publishing review workflows.

## What this example proves

This pack demonstrates the Faye delivery chain:

1. raw user need;
2. simulated materials and directory notes;
3. Faye missing information list;
4. Faye reasonable assumptions;
5. Faye Delivery Pack;
6. Faye `PAL_TEAM.md`;
7. Faye `FAYE_BUILD_REQUEST.md`;
8. PalSmith creates or proposes a Pal team skeleton;
9. optional first real Task Package.

## Boundary

This pack is no-code. It is not a connector, scanner, validator, CLI, marketplace, installer, daemon, database, external API client, auto-sync engine, or auto-execution engine.

Candidate Pals are judgement inputs only. They are not a route map, not central roster edits, and not automatic task assignments.

No credentials, customer-private material, real account data, or external project `.agentpal/` write payload belongs in this reusable example.

## How to use

Use `DELIVERY.md` for the delivery story, `PROJECT.md` for project facts, `FLOWS/` for process shells, `TASKS/` for work packages, and `FAYE_BUILD_REQUEST.md` for the Faye-to-PalSmith handoff.
