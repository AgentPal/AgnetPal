# Integration Notes

## Purpose

This directory records business-system handoff notes for the Video Growth Delivery Pack.

## Profile notes only

Feishu, cloud drive, and publishing platforms may appear as profile or handoff notes. They are not connectors in this example.

This pack does not include:

- real API tokens;
- connector configuration;
- OAuth setup;
- platform API calls;
- account automation;
- cloud drive sync;
- Feishu bot setup;
- publishing automation.

## Manual handoff examples

- Feishu: a human may copy approved task summaries into a meeting note.
- Cloud drive: a human may store approved source material in a private folder.
- Publishing platforms: a human may manually post approved content from the publishing handoff checklist.

## Boundary

No integration note in this pack authorizes external calls, credential storage, or external project `.agentpal/` writes.
