# Expected Output Map

This walkthrough may produce public-safe draft outputs only.

| Output | Safe Location | Notes |
| --- | --- | --- |
| orientation notes | conversation or local scratch outside public repo | do not commit private notes |
| Context Packet draft | conversation or approved public-safe example | no real customer data |
| Delivery Pack outline | conversation or approved public-safe example | fake project only |
| Pal Team Blueprint draft | conversation or approved public-safe example | recommended Pals are judgement inputs |
| verification checklist | conversation or approved public-safe example | keep `missing` and `not-run` explicit |

For a real private project, approved records belong under:

```text
workspace/projects/<project-id>/
```

Do not store private facts in reusable examples.
