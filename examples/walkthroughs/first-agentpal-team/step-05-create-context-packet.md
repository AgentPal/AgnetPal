# Step 05: Create A Context Packet

Use a fake goal:

```text
Create a public-safe content operations AI team outline for demo-content-team.
```

Safe prompt:

```text
Mira, create a Context Packet for this fake goal. Include only public-safe
context from the combined pack and video-growth delivery pack. Do not use real
customer data.
```

Expected Context Packet sections:

- goal
- public-safe sources
- missing information
- assumptions
- private data boundary
- candidate Pal perspectives as AI judgement inputs
- verification needs
