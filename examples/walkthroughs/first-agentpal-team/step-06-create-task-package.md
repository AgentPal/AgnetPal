# Step 06: Create A Task Package

Safe prompt:

```text
Mira, judge whether Faye should create a first Task Package for the fake
demo-content-team Delivery Pack. Keep it no-code and public-safe.
```

Expected Task Package sections:

- task goal
- source scope
- allowed public examples
- forbidden private inputs
- steps
- expected output
- acceptance criteria
- verification evidence

The Task Package should not call APIs, publish content, scan systems, install
software, or create external project folders.
