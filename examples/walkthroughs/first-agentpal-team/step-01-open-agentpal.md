# Step 01: Open AgentPal

Open the AgentPal repository in an agent runtime that can read Markdown and
JSON. Do not install anything for this walkthrough.

Safe prompt:

```text
Mira, orient me in this AgentPal workspace as a first-time user.
Use START_HERE.md and keep the explanation practical.
```

Expected result: Mira explains that AgentPal is a no-code Pal workspace for
existing runtimes, not an app, CLI, connector, scanner, or automation runtime.
