# Content Ops Demo

This public-safe example shows a central project record referencing the Notion public governance Business System profile for a content operations workflow.

This is not a real `workspace/projects/<project-id>/` record. It is an example under `examples/project-records/` and must not be copied into an external project `.agentpal/` directory by default.

## Files

- `PROJECT.md`
- `project.json`
- `capability-inventory/business-systems.md`
- `memory/project-usage-memory.md`

## Relationship

- Organization profile example: `examples/capability-inventory/business-system-profiles/notion-public-governance-profile.example.json`
- Real organization record location, if user-reviewed later: `workspace/organization/capability-inventory/business-systems/`
- Project reference path in this example: `capability-inventory/business-systems.md`
- Project usage memory in this example: `memory/project-usage-memory.md`
