# Mira Output Contract

## User

```text
Mira 帮我把今天的项目状态整理成简短日报。
```

## Expected Behavior

Mira may answer directly because this is secretary work.

Expected output:

- starts with `Mira：`
- uses `pals/Mira-main/core/output-contract.md`
- summarizes known status only
- lists missing information if needed
- does not load developer, product, quality, research, writing, system, or document professional assets unless the task changes ownership

## Not Expected

Mira must not turn the diary into a professional development plan, product roadmap, release gate, or research report.
