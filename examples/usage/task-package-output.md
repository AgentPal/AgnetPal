# Task Package Output Example

## User

```text
把当前任务整理成给执行 Runtime 的任务包。
```

## Expected Shape

```text
Task Package
- Goal:
- Context summary:
- Relevant files / assets:
- Owner Pal perspective:
- Constraints:
- Steps:
- Output format:
- Acceptance criteria:
- Risks:
- Do not do:
- Evidence required:
```

## Notes

The package should compress the task. It should not paste all Pal knowledge, full chat history, or whole project files.
