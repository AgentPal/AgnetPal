# Private Boundary

## Public-Safe Example Boundary

This content operations Org Pack is a public-safe example. It must not contain
real customer data, real account names, credentials, unpublished strategies,
private project memory, private reports, or private evidence.

## Keep Out Of This Pack

- Customer names and account identifiers.
- Platform credentials, cookies, tokens, passwords, or API keys.
- Real editorial calendars from a customer.
- Private campaign results.
- Customer-specific business rules.
- Private source documents.
- Private publishing approvals.

## Approved Targets For Private Material

Private material belongs in approved private project records, customer-private
delivery records, or explicitly approved evidence locations. A reusable public
Org Pack may reference the need for those records, but it must not copy their
contents.

## External Project Boundary

Do not copy this Org Pack into an external project `.agentpal/` directory by
default. External projects remain thin-bound.
