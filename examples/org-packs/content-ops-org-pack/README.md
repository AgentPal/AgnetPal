# Content Ops Org Pack Example

This is a public-safe example Org Pack for a content operations AI organization.

It demonstrates how an Org Pack can describe reusable content operations roles,
workflow candidates, governance, verification, and private boundaries without
including real customer data, real accounts, credentials, connector settings,
scanner logic, validator programs, marketplace logic, or keyword routing.

## Files

- `ORG.md`
- `org.json`
- `recommended-pals.md`
- `workflow-map.md`
- `private-boundary.md`

## Boundary

- Recommended Pals are AI judgement inputs only.
- The example does not copy central Pal contacts.
- The example does not modify the central roster.
- The example does not create external project `.agentpal/org-pack/`.
- The example contains no real customer data, credentials, private business
  secrets, or private project memory.
