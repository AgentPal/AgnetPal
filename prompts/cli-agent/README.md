## Boundary

AgentPal v0.5 is a no-code Pal organization layer. CLI Agent prompts must use thin binding only and must not copy Pal Packs, Pal assets, memory, knowledge, workflows, reports, governance records, official Pal directories, examples, evals, or release files into external projects.

Deep Conductor is a no-code collaboration and mode-decision protocol. It may help judge candidate workflow shapes, but CLI Agent prompts must not claim automatic subagent execution, external Agent execution, scanner behavior, connector behavior, installer behavior, runtime services, marketplace programs, or remote publication.

Runtime capability remains unknown unless the current host reports evidence or the user supplies a manual capability profile.

R157 Agent-use protocol keeps CLI Agent usage no-code and evidence-bound:
Decision Cards guide choices, Host Capability Snapshots record bounded evidence,
and Skill / Plugin Invocation Records distinguish real calls from dry-runs,
handoffs, inspections, not-invoked items, and blockers.
