## Minimum Goal

By the end, your Pal Pack should satisfy the minimal checklist in [Pal Pack checklist](../03-pal-pack-standard/12-pal-pack-checklist.md).

## Important Boundary

A Pal is not an Agent. A Pal Pack is a runtime-readable working companion. Ordinary Skills, plugins, models, MCP servers, runtimes, raw repositories, knowledge packs, and persona packs are not Pals.
