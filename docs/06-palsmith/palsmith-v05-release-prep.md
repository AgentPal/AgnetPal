# PalSmith v0.5 Release Prep

This page is a release-preparation note for the PalSmith v0.5 user-facing path. It is not a release announcement, not a GitHub publishing record, and not evidence that a remote release has been completed.

## Current Scope

PalSmith v0.5 is ready to be reviewed as a no-code Pal asset governance path.

It covers:

- draft Pal and Pal Team design
- composite Pal creation from role, thinking style, voice, knowledge, Skill, plugin, Agent capability, or authorized source-library inputs
- draft Pal Pack review and repair planning
- draft-to-user-custom Pal installation planning
- user custom Pal discovery, invocation, limited delegation, contacts registration, and Marketplace draft authorization design
- revocation planning for user custom Pal discovery
- bounded Runtime Task Packages for approved host-runtime writes
- Quinn-style review evidence for quality, privacy, and boundary checks

## What This Prep Package Does Not Claim

This package does not claim that PalSmith is:

- a CLI
- an installer
- a scanner
- a daemon
- a connector
- an app runtime
- a Marketplace backend
- a deterministic validator program
- an automatic central roster mutator
- an automatic GitHub publishing system

Any file read, write, copy, archive, tag, push, or GitHub Release action remains host-runtime work and needs explicit user authorization plus runtime evidence.

## Release Candidate Readiness

R192 closed the PalSmith v0.5 phase as `ready_for_release_prep`. R193 packages that result into a readable prep set:

- user-visible capability summary
- known limits
- release checklist
- release-prep review evidence
- boundary pressure tests
- Quinn review notes

The package is suitable for user review before any remote publishing decision.

## Publication Boundary

Remote publication is a separate task.

Before any remote action, the user must explicitly authorize:

- `git push`
- tag creation
- tag push
- GitHub Release creation
- release asset upload

Without that authorization, PalSmith should only prepare docs, checklists, evidence, and local reports.

## Recommended User-Facing Summary

PalSmith v0.5 helps users turn goals and reviewed source material into bounded Pal designs, draft Pal Packs, user custom Pal installation plans, authorization records, and review evidence. It keeps private user Pals private by default, separates discovery from delegation and contacts registration, and does not replace the host Agent Runtime.
