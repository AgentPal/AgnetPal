# AgentPal Glossary

## Agent Runtime

The host tool that performs real execution, such as Codex, Claude Code,
OpenCode, OpenHands, or a similar agent runtime.

## AgentPal

A no-code AI organization framework for existing agent runtimes. It is a
Markdown / JSON / protocol workspace, not an app or automation runtime.

## Pal

A working companion made of identity, knowledge, skills, memory rules,
workflows, context rules, output contracts, and verification habits.

## Mira

The default Main Pal, Leader Pal, and Conductor. Ordinary tasks start with Mira.

## PalSmith

The no-code Pal asset governance Pal for creating, reviewing, improving, and
versioning Pal assets and Pal team blueprints.

## Faye

The AI Delivery Pal pattern for turning business goals into Delivery Packs,
Pal Team Blueprints, Faye Build Requests, first Task Packages, acceptance notes,
training notes, and review records.

## Org Pack

A reusable no-code AI organization package.

## FDE Pack

A reusable expert delivery package.

## Delivery Pack

A no-code package that describes how one business goal should be delivered.

## Thin Binding

A small set of external project files that point back to the AgentPal
Workspace. Thin binding does not copy the whole AgentPal workspace.

## Central Project Record

The default AgentPal workspace location for project memory, source maps,
derived knowledge, task records, reports, governance notes, and capability
inventory.

## Context Packet

A compact, bounded package of task context for a Pal or runtime.

## Task Package

A structured instruction package for the host runtime or a Pal-mediated stage.

## Deep Conductor

A no-code protocol set for complex coordination and governance. It is not an
automatic runtime engine.

## AI Judgement Input

A recommendation or reference used by the current AI to judge ownership or
workflow. It is not a route map or deterministic dispatch rule.
