# Current Status

AgentPal v0.5 is a Markdown / JSON / protocol workspace for Pal collaboration in existing host runtimes.

## Release Snapshot

- Version line: `v0.5`
- Current release posture: Codex-first release evidence with non-Codex limits clearly marked
- Primary verified host: Codex
- Default Main Pal / Leader / Conductor: Mira
- Official bundled Pals: Mira, Atlas, Vega, Rhea, PalSmith, Quinn, Morgan, Harper, Nova, Faye
- Runtime boundary: AgentPal is a Pal layer, not an execution layer

## What Is Active

- AgentPal workspace initialization for Codex.
- Mira-first ordinary entry.
- `/pal Name` direct calls through central contacts.
- AI owner judgement from the current central Pal contacts.
- Fast Route for clear owner-Pal handoff.
- Task Package, Context Slicing, Asset Loading Budget, and Runtime Response Gate.
- Deep Conductor as no-code collaboration and mode-decision protocol.
- Capability Inventory as visible evidence, manual profile, or runtime-reported profile.
- Cross-Runtime Pal Memory guidance.
- Runtime-installed Skill Orchestration as candidate naming, not automatic invocation.
- Thin binding for external projects.

## Host Evidence Status

| Host / mode | Current status |
| --- | --- |
| Codex | v0.5 Codex-first verified path |
| Claude Code | limited/minimal handoff evidence; full host acceptance is not claimed |
| Generic CLI Agent | compatibility path; broad host acceptance is not claimed |
| OpenCode / Gemini | unavailable in current evidence |
| DeepSeek | experimental/version-help only |
| Plan Mode UI | unavailable in current UI evidence |
| Goal Mode | limited evidence |

## What Is Not Active

- AgentPal-owned execution runtime.
- Automatic multi-Agent runtime behavior.
- Automatic subagent launch.
- Automatic external Agent orchestration.
- Automatic runtime scanning.
- Automatic connector installation or customer-system execution.
- Automatic Runtime Skill execution by AgentPal.
- Desktop app, web app, marketplace, installer, daemon, scanner, validator, hosted service, token meter, or cost calculator.
