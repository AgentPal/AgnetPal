# Pal Manifesto

## Purpose

This document states the working philosophy behind AgentPal.

## Current Status

Short placeholder for v0.1.0-rc.1. It is intentionally brief and should not become marketing copy.

## Core Idea

A Pal is a portable working companion for an agent runtime. It packages a maintained way of working, not just a longer prompt.

AgentPal keeps that working companion readable, versionable, reviewable, and public-safe.
