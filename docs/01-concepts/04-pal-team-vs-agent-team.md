# Pal Team Vs Agent Team

## Purpose

This document explains the difference between multiple Pals and multiple agents.

## Current Status

Short placeholder for v0.1.0-rc.1.

## Current Boundary

AgentPal v0.1.0-rc.1 does not run a multi-agent team. It uses Simple Pal Mode only.

Multiple Pals can exist in the workspace, but they are working assets for the current runtime. Mira may route to an owner Pal, and direct calls can select a Pal, but this is not the same as starting multiple independent agents.
