# Validation And Evidence

This section explains how to read AgentPal validation material for v0.1.0-rc.1.

The main validation evidence for this release candidate is R33 PalBench small-sample smoke validation. It is useful as release smoke evidence, but it is intentionally narrow: it describes observed workflow benefits in six qualitative samples. It is not a statistically significant benchmark and does not compare underlying model capability.

## Current Boundary

For v0.1.0-rc.1, PalBench should be read as smoke validation for AgentPal working methods: Task Packages, Verification Result Records, Context Slicing, Context Access Lists, Mira single entry, Routing Decision Records, and the Fast Route / Deep Conductor boundary.

Do not read it as a model leaderboard, a broad product benchmark, or proof of general performance improvement.
