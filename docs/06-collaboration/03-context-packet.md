# Context Packet

## Purpose

This document will explain Context Packets for bounded handoff and consultation.

## Current Status

Short placeholder for v0.1.0-rc.1.

## Boundary

A Context Packet should share the minimum useful context. It should not dump full chats, full private files, all memory, all project files, or unrelated Pal assets.
