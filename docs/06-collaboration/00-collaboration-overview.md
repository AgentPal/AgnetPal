# Collaboration Overview

## Purpose

This directory explains Pal contacts, mentions, Context Packets, handoff, and collaboration boundaries.

## Current Status

Short placeholder for v0.1.0-rc.1.

## Current Boundary

AgentPal v0.1.0-rc.1 uses Simple Pal Mode only. Collaboration means same-response routing, handoff, consultation, or structured context sharing through Pal assets. It does not mean active multi-agent execution.
