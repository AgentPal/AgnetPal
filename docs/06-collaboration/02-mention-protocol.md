# Mention Protocol

## Purpose

This document explains `/pal Name` and `@Name` calls.

## Current Status

Short placeholder for v0.1.0-rc.1.

## Basic Rule

Use display names and aliases, not directory names.

Example:

```text
/pal Atlas
```

not:

```text
/pal Atlas-developer
```
