## Boundary

v0.3 is a no-code orchestration prototype line. It may define Multi-Pal Collaboration, Context Packet, Owner + Verifier, Plan -> Execute -> Verify, Parallel Independent Review, Deep Conductor patterns, Pal Skill / Runtime Skill separation, Runtime Skill-aware Task Packages, cross-runtime Pal memory, Token / Cost-aware policy, and Routing Memory writeback protocols.

It does not make AgentPal a runtime, automatic Deep Conductor, Subagent Mode host, external Agent caller, CLI, installer, scanner, validator, daemon, service, database, desktop app, or web app.
