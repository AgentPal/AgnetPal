# File Purpose Index

This page gives a short purpose map for common AgentPal files.

For the detailed navigation map, use root [`RESOURCE_INDEX.md`](../../RESOURCE_INDEX.md).

## Root Files

| File | Purpose |
| --- | --- |
| `README.md` | Public English overview |
| `README.zh-CN.md` | Public Chinese overview |
| `AGENTS.md` | Runtime-facing workspace instructions |
| `PAL.md` | AgentPal workspace Pal-layer entry description |
| `SKILL.md` | Workspace-level skill/navigation entry |
| `agentpal.json` | Workspace metadata and evidence index |
| `RESOURCE_INDEX.md` | Detailed resource navigation |

## Main Directories

| Directory | Purpose |
| --- | --- |
| `core/` | Core gates and shared contracts |
| `prompts/` | Copyable runtime initialization and binding prompts |
| `docs/` | User and integrator documentation |
| `official/pals/` | Official bundled Pal Packs |
| `workspace/organization/contacts/` | Central Pal contacts |
| `templates/` | Reusable templates |
| `evals/` | Evaluation and evidence records |
| `release/` | Release checks and integration notes |

## Boundary

This index is navigation only. It does not authorize full-repository loading or external project writeback.
