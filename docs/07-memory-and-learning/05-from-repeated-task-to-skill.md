# From Repeated Task To Skill

## Purpose

This document explains the repeated-task Skill creation rule.

## Current Status

Short placeholder.

## Current Rule

If the user explicitly asks to save a method as a Skill, or similar operations happen more than three times, the owner Pal should create a formal Pal-owned Skill under that Pal's own `skills/` directory.
