# Create Your First Pal In 5 Minutes

## Purpose

This document will become the shortest path for creating a small Pal Pack.

## Current Status

Short placeholder for v0.1.0-rc.1.

## Basic Idea

Create a directory under `pals/<Name-role>/`, then add the minimum files:

- `PAL.md`
- `pal.json`
- `SKILL.md`
- `README.md`
- `AGENTS.md`
- `core/output-contract.md`
