# Publish Your Pal Pack

## Purpose

This document explains the safe path for adding or sharing a Pal Pack.

## Current Status

Short placeholder based on the previous `Add Pals to pals/` document.

## Basic Path

1. Create a valid Pal Pack directory under `pals/<Name-role>/`.
2. Include the minimum required files.
3. Review the public/private boundary.
4. Add or refresh contacts and registry only after the directory is a valid Pal Pack.
5. Do not add Skills, tools, models, plugins, MCP servers, raw repositories, or persona packs as Pal contacts.
