AgentPal v0.1.0-rc.1 is a release candidate for the AgentPal Workspace and Pal Pack Standard practice.

## What This Release Candidate Means

This release candidate is suitable for public review of:

- the AgentPal Workspace structure
- the official bundled Pal Pack set
- Simple Pal Mode instructions
- contacts and registry source-of-truth rules
- context slicing, asset loading, and task-package protocols
- public-safe release boundaries
- R33 PalBench small-sample smoke validation

It should not be described as a finished app, hosted service, multi-agent runtime, marketplace, installer, daemon, validator, scanner, or execution layer.
