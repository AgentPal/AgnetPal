# Release Scope

AgentPal v0.1.0-rc.1 releases the current Pal Workspace and Pal Pack Standard practice.

## Included

- root initialization and workspace files
- English and Chinese README entry points
- AgentPal workspace metadata
- contacts and registry files for Pal discovery
- eight official bundled Pal Packs
- Simple Pal Mode rules
- Runtime Response Gate and Pal Mode validity rules
- context slicing, asset loading budget, memory boundary, Task Package, and verification protocols
- public examples, evals, prompts, templates, and release notes
- R33 PalBench small-sample smoke validation
- release audit summaries under `release/v0.1.0-rc.1/`

## Not Included

- desktop app
- web UI
- CLI runtime requirement
- daemon, service, scanner, validator, installer, or background process
- automatic execution layer
- active multi-agent runtime
- active subagent orchestration
- active Deep Conductor execution
- marketplace or hosted Pal Hub
- private user memory, private project state, customer data, secrets, or internal development reports

## Future Design Boundary

Some repository material describes future methodology: Deep Conductor, Capability Inventory, Context Access List, Workflow Topology, Routing Reward Memory, PalBench expansion, and possible runtime adapters.

For v0.1.0-rc.1, those materials are design foundations. They do not change the active runtime path.
