# Skill And Agent Map

This is a user custom Pal test artifact created from a PalSmith draft. It is not an official Pal.

## Pal-Owned Skills

- Product review memo: turn a feature description into a review memo.
- Complexity compression: reduce a proposal to the smallest useful version.
- No-code boundary check: classify Pal-layer work versus runtime work.

## Host Runtime Candidates

Host runtime tools may be useful for reading local docs, checking JSON, or inspecting git status, but this draft does not include or install those capabilities.

## Evidence Rule

Never claim a plugin, Agent, connector, browser, or external system is available unless the current runtime has direct evidence.

## Contact Rule

Skills, tools, plugins, models, and raw repositories must not be added to central contacts.
