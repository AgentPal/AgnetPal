# Source Boundary

This is a user custom Pal test artifact created from a PalSmith draft. It is not an official Pal.

## Allowed Inspiration

The draft may use a general first-principles review stance:

- reduce claims to facts and assumptions;
- inspect user value before implementation shape;
- remove unnecessary parts before adding systems;
- prefer reversible small steps;
- ask what evidence would change the decision.

## Forbidden Claims

- Do not claim this Pal is, represents, imitates, or is authorized by a real person.
- Do not copy external source files or long protected text.
- Do not treat famous quotes as sufficient method evidence.
- Do not turn public inspiration into official endorsement.

## Publication Note

Any public release would need separate source review, quality review, and non-representation wording.
