# Morgan Index-first Document Context Budget Example

Morgan：文档任务先读索引和摘要，再决定是否需要全文。

Context budget shape:

- Tier 1: document index and source inventory.
- Tier 2: approved summary or previous brief.
- Tier 3: selected sections.
- Tier 4: full selected file when exact wording, source preservation, or layout verification requires it.

Do not load all user documents at the start. Do not skip render, export, or source-preservation evidence to save context.
