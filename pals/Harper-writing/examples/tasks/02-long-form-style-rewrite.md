# Long-Form Style Rewrite

## User Request

"Rewrite this long article to sound calmer and clearer, but keep the meaning intact."

## When this Pal is a good candidate

Harper is a good candidate because the task is style, voice, structure, and content-preserving rewrite.

## When this Pal should not take the whole task

Harper should not fact-check unfamiliar claims unless source review is included. Vega may be a research candidate for claim verification.

## Task Judgement

- domain_focus: long-form rewriting.
- deliverables: revised draft and change notes.
- work_stages: identify voice target, preserve meaning, rewrite, self-review.
- capability_needs: style control, clarity, content preservation.
- pal_candidates: Harper for writing; Vega candidate if fact verification is requested.
- runtime_candidates: document-editing Runtime if file edits are requested.
- verification_needs: compare meaning, preserve constraints, flag uncertain claims.

## Context Needs

Need original text, target voice, audience, non-negotiable phrases, and allowed degree of restructuring.

## Suggested Task Package

- Goal: rewrite without changing meaning.
- Output: revised text plus notes on major changes and uncertain claims.
- Do not do: remove constraints for smoother prose.

## Good Response Example

Harper improves flow and tone while preserving core claims, caveats, and required wording.

## Bad Response Example

Harper rewrites so aggressively that constraints disappear or the author's position changes.

## Verification / Acceptance Criteria

- Meaning is preserved.
- Style target is visible.
- Changed emphasis is explained when material.
