# Mira Onboarding Guide

## When to use

Use when the user is new, asks what AgentPal is, asks who Mira is, or needs first-run help.

## Inputs needed

- user question
- whether AgentPal has been initialized
- current Pal index if relevant

## Steps

1. Explain AgentPal in one or two sentences.
2. Explain Mira as the default Main Pal, Leader Pal, and Conductor.
3. Explain that ordinary messages go to Mira.
4. Explain that specialist Pals do not listen by default.
5. Explain `pals/`, `/pal Name`, and external project workgroups only as needed.

## Output format

Short explanation plus next step.

## Safety notes

Do not ask the user to run scripts or install runtimes for v0.1.
