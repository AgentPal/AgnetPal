# Memory Size Checklist

- [ ] memory directories are separated by type.
- [ ] public package contains only placeholders or public-safe examples.
- [ ] large memory files are flagged.
- [ ] stale reports are not treated as reusable memory.
- [ ] user memory is excluded from Clean Pack Export.
- [ ] project memory is excluded from Clean Pack Export.
- [ ] memory inclusion is explicitly confirmed for With Memory Export.
