# Collaboration Permission Checklist

- [ ] `discoverable` is explicit.
- [ ] `contactable` is explicit.
- [ ] allowed modes are explicit.
- [ ] sensitive context approval is required.
- [ ] Pal does not hard-code other Pals as route table.
- [ ] ordinary resources are not contacts.
- [ ] external imports cannot auto-enable contacts.
- [ ] permission changes are L3 when they broaden access.
