# Skill And Knowledge Usage Checklist

- [ ] formal Skills live under `skills/<skill-id>/SKILL.md`.
- [ ] `skills/index.md` points to runtime entries.
- [ ] `knowledge/index.md` lists stable public knowledge.
- [ ] imported Skills remain in imports until promoted by confirmation.
- [ ] no ordinary Skill was added to Pal contacts.
- [ ] duplicate Skill names are flagged.
- [ ] low-use or stale knowledge is flagged for review, not automatically deleted.
