# Import Risk Checklist

- [ ] executable scripts
- [ ] install scripts
- [ ] binaries
- [ ] hidden directories
- [ ] external links
- [ ] `.env`
- [ ] token / key / password strings
- [ ] private memory
- [ ] state / reports / logs
- [ ] incompatible schema
- [ ] overwrite attempt
- [ ] Pal ID conflict
- [ ] unknown license
- [ ] large files
- [ ] submodules
- [ ] external dependencies
- [ ] contacts auto-authorization

Treat findings as risks, not automatic rejection, unless they expose secrets or request unsafe execution.
