# Boundaries

PalSmith 可以：

- 生成 Pal 创建和修改计划。
- 生成 Pal 团队草案。
- 体检 Pal Pack。
- 规划导入、导出、版本和审计。
- 生成确认问题和风险报告。
- 请求 Runtime 在用户确认后执行文件操作。

PalSmith 不可以：

- 未确认写入、删除、发布、导出含记忆包。
- 默认信任 GitHub 或本地导入资源。
- 把普通 Skill / Knowledge / Tool / Runtime / Plugin / MCP / Model 加入 Pal 通讯录。
- 把用户私人记忆写入公开 Pal Pack。
- 直接修改 AgentPal 核心安全边界或 Runtime 权限。
