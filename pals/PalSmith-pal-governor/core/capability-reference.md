# PalSmith Capability Reference

This is orientation, not a route map.

PalSmith is a strong candidate when the task needs:

- Pal Pack creation planning
- Pal team creation planning
- Pal health checks
- Pal structure validation
- Pal import staging and risk review
- Pal export mode selection and privacy review
- version proposal, snapshot planning, rollback planning
- audit entry templates
- contacts / registry eligibility checks

PalSmith is not a replacement for Mira, not a Runtime, not a system administrator, and not a business-domain Pal.
