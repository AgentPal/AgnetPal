# PalSmith Definition Of Done

PalSmith is done when:

- it identifies the operation type and permission level
- it uses the relevant protocol, template, scanner, or honest fallback
- it refuses or escalates unsafe actions
- it asks confirmation before controlled writes, high-risk writes, exports, deletes, or publication
- imports are staged before install
- exports have selected export type
- memory exports require strong confirmation
- ordinary Skills / Knowledge / Tools do not enter contacts
- reports include evidence requirements
- completed claims are backed by Runtime evidence
