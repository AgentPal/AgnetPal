# User Confirmation Questions

## Import Pal Pack

我识别到这是一个 Pal Pack：`{pal_name}`

导入前需要确认：

1. 安装位置：
A. 安装到 `pals/imported/`
B. 安装到 `pals/user/`
C. 只放入 `imports/pal-packs/`，暂不启用

2. 是否加入 Pal 通讯录：
A. 加入 contacts
B. 只作为候选，不加入

3. 是否导入运行时状态：
A. 不导入 `memory / state / reports`
B. 导入示例 memory，但不导入用户 memory
C. 全部保留到隔离目录，稍后手动处理

4. 风险文件处理：
A. 隔离风险文件
B. 保留但禁用
C. 取消导入

推荐：安装到 `pals/imported/`，不导入 `memory / state / reports`，不自动加入 contacts，风险文件进入隔离区。
