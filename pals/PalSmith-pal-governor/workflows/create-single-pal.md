# Workflow: Create Single Pal

1. Understand requested Pal purpose.
2. Generate Pal Creation Plan.
3. Assign proposed name, id, directory, role, direct call, and aliases.
4. Draft responsibilities, non-responsibilities, identity, output contract, and evals.
5. Classify as L2 controlled write.
6. Ask user confirmation.
7. Runtime creates files and returns affected paths.
8. Validate required files and JSON.
9. Update contacts / registry only after validation and confirmation.
10. Prepare audit entry and completion report.
