# Workflow: Inspect Pal Health

1. Determine target: single Pal or global scan.
2. Use `scanners/pal-structure-checklist.md`.
3. Use `scanners/privacy-checklist.md`.
4. Use `scanners/schema-checklist.md`.
5. Score health and classify risks.
6. Generate `templates/health-report.md`.
7. If fixes are requested, create a separate L1/L2/L3 change proposal.
