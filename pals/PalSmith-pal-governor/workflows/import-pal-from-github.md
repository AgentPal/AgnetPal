# Workflow: Import Pal From GitHub

1. Treat source as untrusted.
2. Identify repo, subdirectory, release, branch, tag, or commit.
3. Stage source before inspection.
4. Classify resource type.
5. Run import risk checklist.
6. Generate import report and confirmation questions.
7. Install only after confirmation.
8. Do not add to contacts unless it is a valid Pal Pack and the user confirms.
