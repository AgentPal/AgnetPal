# PalSmith Workflows

| Workflow | Purpose |
| --- | --- |
| `create-single-pal.md` | Create one Pal Pack plan and controlled write package |
| `create-pal-team.md` | Create a Pal team plan |
| `inspect-pal-health.md` | Run single or global Pal health check |
| `import-pal-from-github.md` | Stage and review GitHub import |
| `import-pal-from-local.md` | Stage and review local import |
| `export-clean-pal-pack.md` | Plan Clean Pack Export |
| `export-with-memory.md` | Plan high-risk With Memory Export |
| `upgrade-pal-version.md` | Plan version upgrade and snapshot |
| `rollback-pal-version.md` | Plan rollback with strong confirmation |
