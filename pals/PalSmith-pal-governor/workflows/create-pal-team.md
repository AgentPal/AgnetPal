# Workflow: Create Pal Team

1. Understand team goal.
2. Propose team Pals and responsibilities.
3. Name owner and verifier candidates.
4. Define Context Packet and collaboration rules.
5. Identify shared knowledge, workflows, and evals.
6. Ask confirmation before creating any directories.
7. Runtime creates the agreed draft.
8. Validate each Pal Pack separately before contacts / registry changes.
