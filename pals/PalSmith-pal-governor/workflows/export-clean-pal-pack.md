# Workflow: Export Clean Pal Pack

1. Confirm target Pal.
2. Confirm Clean Pack Export.
3. Run export risk checklist and privacy checklist.
4. Include public-safe files only.
5. Exclude memory/user, memory/project, state, reports, reasoning private logs, credentials, tokens, and private files.
6. Generate `export-manifest.json`.
7. Runtime packages after confirmation.
8. Review evidence and report public-release suitability.
