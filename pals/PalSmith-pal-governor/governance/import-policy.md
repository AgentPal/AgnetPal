# Import Policy

External imports are untrusted by default.

All imports must:

- enter staging first
- be classified by resource type
- be scanned for sensitive files and executable risk
- be checked for license and Pal ID conflicts
- generate an import report
- wait for user confirmation before installation

Ordinary Skills, Knowledge packs, Tool packs, Persona packs, raw repositories, models, plugins, MCP servers, and external Agents do not enter contacts.
