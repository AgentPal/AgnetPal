# Deep Conductor Agent-use Mode Decision Protocol

Status: v0.5 no-code protocol

Deep Conductor mode decisions must use the R157 Agent-use standards when a task
involves host runtime choice, Codex mode choice, model/reasoning advice,
Skill/plugin candidates, subthread/subagent candidates, or external Agent
handoff.

Required references:

- `standards/agent-use/agent-use-decision-card.md`
- `standards/capability-inventory/host-capability-snapshot.md`
- `standards/agent-use/codex-mode-selection-matrix.md`
- `standards/agent-use/model-reasoning-recommendation.md`
- `standards/agent-use/subthread-subagent-decision.md`
- `standards/agent-use/skill-plugin-invocation-record.md`

Deep Conductor remains no-code. These standards do not start Pals, create
subagents, call plugins, scan hosts, or execute external Agents.
