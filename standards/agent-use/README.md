# Agent-use Standards

Status: v0.5 no-code protocol

This directory defines how AgentPal Pals help a user decide how to use a host
Agent runtime, model, reasoning level, Skill, plugin, subthread, subagent, or
external Agent handoff.

These standards are planning and evidence protocols. They do not install,
scan, invoke, publish, sync, or execute anything by themselves.

Core entries:

- `agent-use-decision-card.md`
- `skill-plugin-invocation-record.md`
- `codex-mode-selection-matrix.md`
- `model-reasoning-recommendation.md`
- `subthread-subagent-decision.md`
- `claude-code-handoff-acceptance-card.md`
- `pal-pre-execution-self-question-checklist.md`
