# Current Release Pointers

Current public release line: `v0.3.0-rc.1`.

Root release files remain available for conventional repository readers:

- `RELEASE_NOTES.md`
- `CHANGELOG.md`
- `RELEASE_CHECKLIST.md`
- `RELEASE_MANIFEST.md`
- `GITHUB_RELEASE_DRAFT.md`

Historical release directories stay under `release/history/` or versioned `release/<version>/` directories. Fresh-clone and audit records may be collected under `release/audits/` and `release/fresh-clone-checks/`.
