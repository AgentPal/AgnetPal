# AgentPal v0.5 Release Notes Draft After Faye Extension

Status: draft only. Not published. No GitHub Release has been created.

## Summary

AgentPal v0.5 expands the public no-code organization layer around Pal teams,
Pal assets, Org Packs, FDE Packs, Delivery Packs, PalSmith governance, and
release-quality evidence.

This refreshed draft adds the Faye / Delivery Pack / Deep Conductor extension
after R136 targeted regression passed.

## Highlights

- Adds Faye / AI Delivery Pal as a no-code delivery-owner pattern.
- Adds Delivery Pack standard and base Delivery Pack template.
- Adds public-safe Video Growth Delivery Pack example.
- Adds public-safe Sales CRM Delivery Pack example.
- Adds Faye -> PalSmith handoff standard.
- Adds Deep Conductor Delivery Pack governance loop, Context Access List,
  templates, and examples.
- Adds R136 targeted regression evidence with 8 / 8 groups passed and 0
  blocker/high/medium issues.
- Keeps Simple Pal Mode as the current runtime policy.
- Keeps AgentPal as a Markdown / JSON / protocol workspace, not an execution
  layer.

## Existing v0.5 Highlights Retained

- Org Pack standard, practical Org Pack template, and Org/FDE combined usage
  guidance.
- FDE Pack base template and public-safe combined pack example.
- Asset Boundary standards for reusable public assets versus private customer
  or project assets.
- Pal Asset standards and Pal Metadata v0.5 upgrade planning.

  guidance.
- R127-R131 v0.5 regression, completion, and release-preflight evidence.
