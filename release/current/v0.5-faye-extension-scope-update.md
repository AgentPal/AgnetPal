# AgentPal v0.5 Faye Extension Scope Update

Status: v0.5 scope extended after local completion.

R130 and R131 remain valid for the pre-Faye-extension v0.5 package scope. After
R132, R133, and R134, the v0.5 public package now includes Faye / Delivery Pack
/ Deep Conductor assets that were not covered by the R130 completion claim or
the R131 local release preflight.

## Scope Change

The v0.5 package now includes:

- R132-A Faye / AI Delivery Pal standards, Delivery Pack standard, handoff
  standard, base template, and base example shell.
- R132-B video growth Delivery Pack example.
- R132-C sales CRM Delivery Pack example.
- R132-D Faye Deep Conductor protocol, governance loop, Context Access List
  standard, templates, and public-safe examples.
- R133 shared entry and navigation integration.
- R134 review gate evidence and safe documentation fix.

## Completion Status Impact

R130 local completion is not deleted or invalidated. It is now a historical
completion statement for the pre-Faye-extension scope.

R131 local release preflight is not deleted or invalidated. It is now a
historical local preflight statement for the pre-Faye-extension scope.

A refreshed final v0.5 release-ready claim must wait until R136 targeted
regression covers the Faye extension.

## Unchanged Boundaries

This scope update does not change the AgentPal product boundary:

- AgentPal remains a Markdown / JSON / protocol workspace.
- Current runtime policy remains Simple Pal Mode.
- The central roster remains 9 official bundled Pals.
- No official Pal is added by this extension.
- PalSmith remains the only official Pal with an asset manifest.
- Owner selection remains AI judgement only.
- Keyword routing remains disallowed.
- External projects still use thin binding only.
- The extension is no-code organization and delivery methodology, not an
  execution runtime.

## Not Added

R132-R135 do not add:

- CLI, web app, desktop app, installer, daemon, database, connector, scanner,
  validator, marketplace program, runtime resolver, or auto-execution engine;
- automatic multi-agent execution;
- external project `.agentpal` writes;
- official Pal roster changes;
- central contact changes;
- customer-private data or credentials;
- git push, pull, fetch, tag creation, GitHub Release, or remote publication.

## R136 Requirement

R136 should run targeted regression over the Faye extension before any refreshed
v0.5 final completion, release-ready, or remote-publication readiness claim.
