# Release Audits

This directory stores public-safe release evidence for AgentPal release candidates and stable releases.

Release audit files are not private task reports. They should contain only public-safe summaries, checks performed, findings, fixed issues, remaining risks, and release recommendations.

## Current Release

- [v0.1.0-rc.1](v0.1.0-rc.1/)

## Maintenance Rule

Create a new versioned directory for each release candidate or stable release. Do not reuse old audit conclusions after repository content changes.
